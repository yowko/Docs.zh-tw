 <span data-ttu-id="aebdb-137">中繼資料系統會針對附加至訊息、作業與端點原則主體的每個原則替代項目組合，在所有已註冊的原則匯入延伸上重複呼叫 <xref:System.ServiceModel.Description.IPolicyImportExtension.ImportPolicy%2A?displayProperty=nameWithType> 方法。</span><span class="sxs-lookup"><span data-stu-id="aebdb-137">The metadata system repeatedly calls the <xref:System.ServiceModel.Description.IPolicyImportExtension.ImportPolicy%2A?displayProperty=nameWithType> method on all registered policy import extensions for each combination of policy alternatives attached to the message, operation, and endpoint policy subjects.</span></span> <span data-ttu-id="aebdb-138">一旦匯入 WSDL 連接埠，附加至連接埠與對應 WSDL 繫結的原則會在呼叫原則匯入延伸之前合併。</span><span class="sxs-lookup"><span data-stu-id="aebdb-138">When importing a WSDL port, policies attached to the port and to the corresponding WSDL binding are merged before calling into the policy import extensions.</span></span> <span data-ttu-id="aebdb-139">原則替代項目會透過 <xref:System.ServiceModel.Description.PolicyConversionContext> 以 <xref:System.ServiceModel.Description.PolicyAssertionCollection> 物件形式來提供。</span><span class="sxs-lookup"><span data-stu-id="aebdb-139">The policy alternatives are made available through a <xref:System.ServiceModel.Description.PolicyConversionContext> as <xref:System.ServiceModel.Description.PolicyAssertionCollection> objects.</span></span> <span data-ttu-id="aebdb-140">每一個 <xref:System.ServiceModel.Description.PolicyAssertionCollection> 都是由 <xref:System.Xml.XmlElement> 物件所表示的原則判斷提示集合。</span><span class="sxs-lookup"><span data-stu-id="aebdb-140">Each <xref:System.ServiceModel.Description.PolicyAssertionCollection> is a collection of policy assertions represented by <xref:System.Xml.XmlElement> objects.</span></span>
