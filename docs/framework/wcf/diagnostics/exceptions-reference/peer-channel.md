# <a name="peer-channel"></a>對等通道
這個主題會列出 [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] 對等通道產生的所有例外狀況。  
  
## <a name="exception-list"></a>例外狀況清單  
  
|資源程式碼|資源字串|  
|-------------------|---------------------|  
|InsufficientCredentials|指定的認證不足以執行要求的作業。 請對指定的作業指定有效的值。|  
|InvalidResolverMode|指定的 PeerResolverMode 值無效。 請指定 PeerResolveMode.Auto、Default 或 PNRP。|  
|PeerNodeAborted|因為已經終止 PeerNode，所以無法開啟。|  
|PeerNullRegistrationInfo|登錄資訊不可為 null。 請確定已使用有效的 RegistrationInfo 物件叫用登錄作業。|  
|PeerNullResolveInfo|解析資訊不得為 `null`。 請確定已使用有效的 ResolveInfo 物件叫用解析作業。|
