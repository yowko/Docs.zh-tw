 <span data-ttu-id="26437-110">不建議使用單獨的 IIS 7.0，因為有管理和監視工具，可使用 AppFabric 協助維護執行中應用程式。</span><span class="sxs-lookup"><span data-stu-id="26437-110">Using IIS 7.0 alone is not recommended, as there are management and monitoring tools available with AppFabric that facilitate maintenance of running applications.</span></span> <span data-ttu-id="26437-111">如果有基礎結構考量，移至 AppFabric 的 IIS 7.0 中時，應該只裝載工作流程。</span><span class="sxs-lookup"><span data-stu-id="26437-111">Workflows should only be hosted in IIS 7.0 alone if there are infrastructure concerns with moving to AppFabric.</span></span>  
  
> [!WARNING]
