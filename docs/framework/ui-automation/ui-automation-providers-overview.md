## <a name="types-of-providers"></a><span data-ttu-id="72149-112">提供者類型</span><span class="sxs-lookup"><span data-stu-id="72149-112">Types of Providers</span></span>

### <a name="server-side-providers"></a><span data-ttu-id="72149-118">伺服器端提供者</span><span class="sxs-lookup"><span data-stu-id="72149-118">Server-side providers</span></span>
