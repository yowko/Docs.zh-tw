# <a name="getfiledef-method"></a>GetFileDef 方法
擷取中繼資料 （而不是指派的 ALink 語彙基元） 中所使用的實際 FileDef 權杖。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT GetFileDef(  
    mdAssembly AssemblyID,  
    mdFile TargetFile,  
    mdFile* pScope  
) PURE;  
```  
  
#### <a name="parameters"></a>參數  
 `AssemblyID`  
 組件的識別碼。  
  
 `TargetFile`  
 語彙基元加入的檔案從 AddFile 方法或 AddImport 方法擷取。  
  
 `pScope`  
 接收 FileDef 語彙基元。  
  
## <a name="return-value"></a>傳回值  
 如果方法成功則傳回 S_OK。  
  
## <a name="requirements"></a>需求  
 需要 alink.h
