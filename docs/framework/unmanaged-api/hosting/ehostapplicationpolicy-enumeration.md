# <a name="ehostapplicationpolicy-enumeration"></a>EHostApplicationPolicy 列舉

## <a name="syntax"></a>語法  
  
```

    HOST_APPLICATION_BINDING_POLICY = 1  
} EHostApplicationPolicy;  
```
