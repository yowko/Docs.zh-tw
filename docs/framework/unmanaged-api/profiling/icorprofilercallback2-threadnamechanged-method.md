# <a name="icorprofilercallback2threadnamechanged-method"></a>ICorProfilerCallback2::ThreadNameChanged 方法
通知程式碼剖析工具，執行緒的名稱已變更。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT ThreadNameChanged(  
    [in] ThreadID threadId,  
    [in] ULONG cchName,  
    [in] WCHAR name[]);  
```  
  
#### <a name="parameters"></a>參數  
 `threadId`  
 [in]執行緒的識別碼。  
  
 `cchName`  
 [in]執行緒的新名稱的長度。  
  
 `name`  
 [in]執行緒的新名稱。 名稱不是 null 終止的。
