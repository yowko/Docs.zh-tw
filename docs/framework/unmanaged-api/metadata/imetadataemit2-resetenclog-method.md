# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog 方法
重設的編輯後繼續的記錄檔，並啟動新的工作階段。  
  
## <a name="syntax"></a>語法  
  
```  
HRESULT ResetENCLog ();  
```  
  
#### <a name="parameters"></a>參數  
 `ResetENCLog` 不接受任何參數。
