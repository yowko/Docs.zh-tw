# <a name="metadata-structures"></a>中繼資料結構
本節說明中繼資料 API 所使用的 Unmanaged 結構。  
  
## <a name="in-this-section"></a>本章節內容  
 ASSEMBLYMETADATA  
 儲存組件中繼資料供稍後使用。  
  
 [COR_FIELD_OFFSET 結構](../../../../docs/framework/unmanaged-api/metadata/cor-field-offset-structure.md)  
 儲存指定欄位的位移 (在類別中)。  
  
 [COR_NATIVE_LINK 結構](../../../../docs/framework/unmanaged-api/metadata/cor-native-link-structure.md)  
 包含用來連結原生程式碼的資訊。  
  
 [CVStruct 結構](../../../../docs/framework/unmanaged-api/metadata/cvstruct-structure.md)  
 包含在安裝模組或複合映像時，所使用的資訊。  
  
 [OSINFO 結構](../../../../docs/framework/unmanaged-api/metadata/osinfo-structure.md)  
 包含作業系統的相關資訊。
