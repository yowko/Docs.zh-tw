# <a name="imetadatatablesgetuserstring-method"></a>IMetaDataTables::GetUserString 方法

取得在目前範圍中的字串資料行中的指定索引處的硬式編碼的字串。

## <a name="syntax"></a>語法

```cpp
HRESULT GetUserString (
    [in]  ULONG       ixUserString,
    [out] ULONG       *pcbData,
    [out] const void  **ppData
);
```

## <a name="parameters"></a>參數

`ixUserString`\
[in]硬式編碼的字串將擷取的索引值。

`pcbData`\
[out]大小的指標`ppData`。

`ppData`\
[out]傳回的字串指標的指標。
