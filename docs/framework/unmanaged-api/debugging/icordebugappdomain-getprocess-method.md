# <a name="icordebugappdomaingetprocess-method"></a>ICorDebugAppDomain::GetProcess 方法
取得包含應用程式定義域的處理序。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetProcess (  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppProcess`  
 [out]ICorDebugProcess 物件，表示處理程序的位址指標。
