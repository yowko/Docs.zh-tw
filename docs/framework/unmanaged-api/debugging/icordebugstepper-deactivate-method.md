# <a name="icordebugstepperdeactivate-method"></a>ICorDebugStepper::Deactivate 方法
會導致此 ICorDebugStepper 取消它收到的最後一個步驟命令。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Deactivate ();  
```  
  
## <a name="remarks"></a>備註  
 最近收到的步驟命令已被取消之後，可能會發出新的逐步執行命令。
