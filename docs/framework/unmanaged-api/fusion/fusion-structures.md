# <a name="fusion-structures"></a>融合結構
本節說明融合 API 所使用的非受控結構。  
  
## <a name="in-this-section"></a>本章節內容  
 [ASSEMBLY_INFO 結構](assembly-info-structure.md)  
 包含在全域組件快取中註冊之元件的相關資訊。  
  
 [FUSION_INSTALL_REFERENCE 結構](fusion-install-reference-structure.md)  
 表示應用程式對應用程式安裝在全域組件快取中的元件所做的參考。  
  
 [IDENTITY_ATTRIBUTE 結構](identity-attribute-structure.md)  
 包含有關[IDefinitionIdentity](idefinitionidentity-interface.md)實例的中繼資料屬性資訊。  
  
 [IDENTITY_ATTRIBUTE_BLOB 結構](identity-attribute-blob-structure.md)  
 包含相關聯之[IDENTITY_ATTRIBUTE](identity-attribute-structure.md)結構的位移資訊。
