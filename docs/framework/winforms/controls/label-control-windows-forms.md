# <a name="label-control-windows-forms"></a>Label 控制項 (Windows Form)
> [!IMPORTANT]
> 控制項會取代並將功能<xref:System.Windows.Forms.Label>加入控制項。 <xref:System.Windows.Forms.ToolStripLabel> 您可以搭配其他<xref:System.Windows.Forms.ToolStripLabel>新的控制項使用, 例如。 <xref:System.Windows.Forms.ToolStripDropDown> 不過, <xref:System.Windows.Forms.Label>如果您選擇, 則會保留控制項以提供回溯相容性及未來使用。  
  
 Windows Forms <xref:System.Windows.Forms.Label>控制項是用來顯示使用者無法編輯的文字或影像。 它們可用來識別表單上的物件, 以提供特定控制項在按一下時會執行的動作, 例如, 或顯示資訊以回應應用程式中的運行時間事件或進程。 <xref:System.Windows.Forms.Label>因為控制項無法接收焦點, 所以它也可以用來建立其他控制項的存取金鑰。
