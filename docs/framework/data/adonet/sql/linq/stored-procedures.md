# <a name="stored-procedures"></a>預存程序
[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]使用物件模型中的方法，表示資料庫中的預存程式。 您可套用 <xref:System.Data.Linq.Mapping.FunctionAttribute> 屬性 (Attribute) 和 (視需要) 套用 <xref:System.Data.Linq.Mapping.ParameterAttribute> 屬性，將方法指定為預存程序。 如需詳細資訊，請參閱[LINQ to SQL 物件模型](the-linq-to-sql-object-model.md)。  
  
 使用 Visual Studio 的開發人員通常會使用物件關聯式設計工具來對應預存程式。 本節中的主題顯示自行撰寫程式碼時，如何在應用程式中形成和呼叫這些方法。

 描述如何使用輸入和輸出參數。
