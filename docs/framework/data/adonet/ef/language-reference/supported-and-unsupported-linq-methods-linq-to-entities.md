# <a name="supported-and-unsupported-linq-methods-linq-to-entities"></a>支援與不支援的 LINQ 方法 (LINQ to Entities)
本節提供 LINQ to Entities 查詢中支援或不支援之語言整合式查詢（LINQ）標準查詢運算子的相關資訊。 許多 LINQ 標準查詢運算子都有一個可接受整數引數的多載版本。 整數引數會對應至在、 <xref:System.Collections.Generic.IEqualityComparer%601>或<xref:System.Collections.Generic.IComparer%601>的序列中以零為基底的索引。 除非另外指定，否則不支援 LINQ 標準查詢運算子的這些多載版本，而且嘗試使用它們將會擲回例外狀況。  
  
## <a name="projection-and-restriction-methods"></a>投影和限制方法  
 大部分的 LINQ 投影和限制方法在 LINQ to Entities 查詢中都受到支援，但接受位置引數的例外狀況除外。 如需詳細資訊，請參閱[LINQ to Entities 查詢中的標準查詢運算子](standard-query-operators-in-linq-to-entities-queries.md)。 下表所列為支援及不支援的投影和限制方法。

|<xref:System.Linq.Queryable.Select%2A>|不支援|`Function Select(Of TSource, TResult) ( _ source As IQueryable(Of TSource), _ selector As Expression(Of Func(Of TSource, Integer, TResult)) _ ) As IQueryable(Of TResult)`|`IQueryable<TResult> Select<TSource, TResult>( this IQueryable<TSource> source, Expression<Func<TSource, int, TResult>> selector )`|  
|<xref:System.Linq.Queryable.SelectMany%2A>|支援|`Function SelectMany(Of TSource, TResult) ( _ source As IQueryable(Of TSource), _ selector As Expression(Of Func(Of TSource, IEnumerable(Of TResult))) _ ) As IQueryable(Of TResult)`|`IQueryable<TResult> SelectMany<TSource, TResult>( this IQueryable<TSource> source, Expression<Func<TSource, IEnumerable<TResult>>> selector )`|
