## <a name="managing-failure-and-providing-resiliency"></a><span data-ttu-id="31cd6-187">管理失敗並提供復原功能</span><span class="sxs-lookup"><span data-stu-id="31cd6-187">Managing failure and providing resiliency</span></span>
