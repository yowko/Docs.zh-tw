### <a name="example-of-using-both-styles"></a><span data-ttu-id="66e99-113">同時使用兩種樣式的範例</span><span class="sxs-lookup"><span data-stu-id="66e99-113">Example of Using Both Styles</span></span>  
 <span data-ttu-id="66e99-114">當您建立 XML Web 服務，可同時使用方法上的兩組屬性。</span><span class="sxs-lookup"><span data-stu-id="66e99-114">When you're creating an XML Web service, you can use both sets of attributes on the methods.</span></span> <span data-ttu-id="66e99-115">在下列程式碼範例中，名為 `MyService``MyLiteralMethod`的類別包含兩個 XML Web 服務方法：`MyEncodedMethod` 及 。</span><span class="sxs-lookup"><span data-stu-id="66e99-115">In the following code example, the class named `MyService` contains two XML Web service methods, `MyLiteralMethod` and `MyEncodedMethod`.</span></span> <span data-ttu-id="66e99-116">兩種方法都執行相同功能：傳回 `Order` 類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="66e99-116">Both methods perform the same function: returning an instance of the `Order` class.</span></span> <span data-ttu-id="66e99-117">在 `Order` 類別中，<xref:System.Xml.Serialization.XmlTypeAttribute> 以及 <xref:System.Xml.Serialization.SoapTypeAttribute> 屬性 (attribute) 都套用至 `OrderID` 欄位，而且兩個屬性 (attribute) 的 `ElementName` 屬性 (property) 都設為不同值。</span><span class="sxs-lookup"><span data-stu-id="66e99-117">In the `Order` class, the <xref:System.Xml.Serialization.XmlTypeAttribute> and the <xref:System.Xml.Serialization.SoapTypeAttribute> attributes are both applied to the `OrderID` field, and both attributes have their `ElementName` property set to different values.</span></span>  
  
 <span data-ttu-id="66e99-118">若要執行這個範例，請將程式碼貼至副檔名為 .asmx 的檔案中，並將該檔置於 Internet Information Services (IIS) 管理的虛擬目錄中。</span><span class="sxs-lookup"><span data-stu-id="66e99-118">To run the example, paste the code into a file with an .asmx extension, and place the file into a virtual directory managed by Internet Information Services (IIS).</span></span> <span data-ttu-id="66e99-119">從 Internet Explorer 這類 HTML 瀏覽器中，輸入電腦、虛擬目錄及檔案的名稱。</span><span class="sxs-lookup"><span data-stu-id="66e99-119">From an HTML browser, such as Internet Explorer, type the name of the computer, virtual directory, and file.</span></span>  
  
```vb  
<%@ WebService Language="VB" Class="MyService" %>  
Imports System  
Imports System.Web.Services  
Imports System.Web.Services.Protocols  
Imports System.Xml.Serialization  
Public Class Order  
    ' Both types of attributes can be applied. Depending on which type  
    ' the method used, either one will affect the call.  
    <SoapElement(ElementName:= "EncodedOrderID"), _  
    XmlElement(ElementName:= "LiteralOrderID")> _  
    public OrderID As String  
End Class  
  
Public Class MyService  
    <WebMethod, SoapDocumentMethod> _  
    public Function MyLiteralMethod() As Order   
        Dim myOrder As Order = New Order()  
        return myOrder  
    End Function  
    <WebMethod, SoapRpcMethod> _  
    public Function MyEncodedMethod() As Order   
        Dim myOrder As Order = New Order()  
        return myOrder  
    End Function  
End Class  
```  
  
```csharp  
<%@ WebService Language="C#" Class="MyService" %>  
using System;  
using System.Web.Services;  
using System.Web.Services.Protocols;  
using System.Xml.Serialization;  
public class Order{  
    // Both types of attributes can be applied. Depending on which type  
    // the method used, either one will affect the call.  
    [SoapElement(ElementName = "EncodedOrderID")]  
    [XmlElement(ElementName = "LiteralOrderID")]  
    public String OrderID;  
}  
public class MyService{  
    [WebMethod][SoapDocumentMethod]  
    public Order MyLiteralMethod(){  
        Order myOrder = new Order();  
        return myOrder;  
    }  
    [WebMethod][SoapRpcMethod]  
    public Order MyEncodedMethod(){  
        Order myOrder = new Order();  
        return myOrder;  
    }  
}  
```  
  
 <span data-ttu-id="66e99-120">下列程式碼範例呼叫 `MyLiteralMethod`。</span><span class="sxs-lookup"><span data-stu-id="66e99-120">The following code example calls `MyLiteralMethod`.</span></span> <span data-ttu-id="66e99-121">項目名稱變更為 "LiteralOrderID"。</span><span class="sxs-lookup"><span data-stu-id="66e99-121">The element name is changed to "LiteralOrderID".</span></span>  
  
```xml  
<?xml version="1.0" encoding="utf-8"?>

    <soap:Body>  
        <MyLiteralMethodResponse xmlns="http://tempuri.org/">  
            <MyLiteralMethodResult>

        </MyLiteralMethodResponse>
```  
