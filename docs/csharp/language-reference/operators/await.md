<span data-ttu-id="6e6c7-103">`await` 運算子會暫停封入[非同步](../keywords/async.md)方法的評估，直到其運算元所代表的非同步作業完成為止。</span><span class="sxs-lookup"><span data-stu-id="6e6c7-103">The `await` operator suspends evaluation of the enclosing [async](../keywords/async.md) method until the asynchronous operation represented by its operand completes.</span></span> <span data-ttu-id="6e6c7-104">當非同步作業完成時，`await` 運算子會傳回作業的結果 (如果有的話)。</span><span class="sxs-lookup"><span data-stu-id="6e6c7-104">When the asynchronous operation completes, the `await` operator returns the result of the operation, if any.</span></span> <span data-ttu-id="6e6c7-105">當 `await` 運算子套用至代表已完成作業的運算元時，它會立即傳回作業的結果，而不會暫停封入方法。</span><span class="sxs-lookup"><span data-stu-id="6e6c7-105">When the `await` operator is applied to the operand that represents already completed operation, it returns the result of the operation immediately without suspension of the enclosing method.</span></span> <span data-ttu-id="6e6c7-106">`await` 運算子不會封鎖評估非同步方法的執行緒。</span><span class="sxs-lookup"><span data-stu-id="6e6c7-106">The `await` operator doesn't block the thread that evaluates the async method.</span></span> <span data-ttu-id="6e6c7-107">當 `await` 運算子暫停封入方法時，控制項會返回方法的呼叫端。</span><span class="sxs-lookup"><span data-stu-id="6e6c7-107">When the `await` operator suspends the enclosing method, the control returns to the caller of the method.</span></span>
