- <span data-ttu-id="97c36-120">使用屬性與 Friend 組件可能會拖累編譯器效率。</span><span class="sxs-lookup"><span data-stu-id="97c36-120">Using attributes with friend assemblies can hamper compiler efficiency.</span></span> <span data-ttu-id="97c36-121">當您使用屬性時，編譯器不知道金鑰為何，但它必須決定是否要授與夥伴關係，因此必須猜測。</span><span class="sxs-lookup"><span data-stu-id="97c36-121">When you use attributes, the compiler does not know what the key is when it has to decide whether or not to grant friendship and so it has to guess.</span></span> <span data-ttu-id="97c36-122">在編譯結束時，編譯器最後知道金鑰之後便能夠確認先前的猜測。</span><span class="sxs-lookup"><span data-stu-id="97c36-122">At the end of compilation, the compiler is able to verify the guess once it finally knows the key.</span></span> <span data-ttu-id="97c36-123">使用編譯器選項指定金鑰檔時，編譯器可以立即判斷是否要授與夥伴關係。</span><span class="sxs-lookup"><span data-stu-id="97c36-123">When the key file is specified with a compiler option, the compiler can immediately decide whether to grant friendship.</span></span>
