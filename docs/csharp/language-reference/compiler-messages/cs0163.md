 <span data-ttu-id="46bf7-104">當 [switch 陳述式](../keywords/switch.md)包含多個 switch 區段時，您必須使用下列其中一個關鍵字明確結束每個區段，包括最後一個區段：</span><span class="sxs-lookup"><span data-stu-id="46bf7-104">When a [switch statement](../keywords/switch.md) contains more than one switch section, you must explicitly terminate each section, including the last one, by using one of the following keywords:</span></span>

 <span data-ttu-id="46bf7-112">下列範例會產生 CS0163。</span><span class="sxs-lookup"><span data-stu-id="46bf7-112">The following sample generates CS0163.</span></span>  
  
```csharp  
// CS0163.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        int i = 0;  
  
        switch (i)   // CS0163  
        {  
            // Compiler error CS0163 is reported on the following line.  
            case 1:  
                i++;  
            // To resolve the error, uncomment one of the following example statements.  
            // return;  
            // break;  
            // goto case 3;  
  
            case 2:  
                i++;  
                return;  
  
            case 3:  
                i = 0;  
                return;  
  
            // Compiler error CS0163 is reported on the following line.  
            default:  
                Console.WriteLine("Default");  
                // To resolve the error, uncomment the following line:  
            //break;  
        }  
    }  
```
