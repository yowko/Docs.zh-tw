# <a name="compiler-error-cs0843"></a>編譯器錯誤 CS0843
程式控制權回到呼叫端前，必須完整指派自動實作的屬性 'name' 的支援欄位。 請考慮從建構函式初始設定式呼叫無參數的建構函式。  
  
 若要從建構函式指派值給自動實作屬性，您必須先叫用無參數的建構函式來建立該物件。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 在建構函式初始設定式中新增無參數建構函式的呼叫，如下列範例所示。 注意 `: this()` 的用法。 如需詳細資訊，請參閱 [this](../keywords/this.md)。  
  
## <a name="example"></a>範例  
 下列程式碼會產生 CS0843：  
  
```csharp  
// cs0843.cs  
struct S  
{  
    public int AIProp { get; set; }  
    public S(int i){} //CS0843  
    // Try the following lines instead.  
   // public S(int i) : this()  
   // {  
   //     AIProp = i;  
   // }  
}  
  
class Test  
{  
    static int Main()  
    {  
        return 1;  
    }  
}  
```
