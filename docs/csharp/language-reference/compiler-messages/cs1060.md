# <a name="compiler-warning-level-1-cs1060"></a><span data-ttu-id="abeb9-102">編譯器警告 (層級 1) CS1060</span><span class="sxs-lookup"><span data-stu-id="abeb9-102">Compiler Warning (level 1) CS1060</span></span>
<span data-ttu-id="abeb9-103">使用可能未指派的欄位 'name'。</span><span class="sxs-lookup"><span data-stu-id="abeb9-103">Use of possibly unassigned field 'name'.</span></span> <span data-ttu-id="abeb9-104">如果未指派結構，則初始時不會指派結構執行個體變數。</span><span class="sxs-lookup"><span data-stu-id="abeb9-104">Struct instance variables are initially unassigned if struct is unassigned.</span></span>  
  
 <span data-ttu-id="abeb9-105">如果未明確初始化結構成員，則會將其初始化為預設值。</span><span class="sxs-lookup"><span data-stu-id="abeb9-105">Struct members are initialized to their default value if you do not explicitly initialize them.</span></span> <span data-ttu-id="abeb9-106">類別類型 (及其他參考型別) 的預設值為 null。</span><span class="sxs-lookup"><span data-stu-id="abeb9-106">The default value for class types (and other reference types) is null.</span></span> <span data-ttu-id="abeb9-107">如果未初始化類別就嘗試對其進行存取，則會在執行階段擲回 `NullReferenceException`。</span><span class="sxs-lookup"><span data-stu-id="abeb9-107">If the class is not initialized before any attempt to access it, a `NullReferenceException` will be thrown at runtime.</span></span> <span data-ttu-id="abeb9-108">編譯器無法明確判斷是否將初始化類別成員，因此 CS1060 是警告，而不是錯誤。</span><span class="sxs-lookup"><span data-stu-id="abeb9-108">The compiler cannot determine definitively whether the class member will be initialized or not, and so CS1060 is a warning and not an error.</span></span>

## <a name="example"></a><span data-ttu-id="abeb9-111">範例</span><span class="sxs-lookup"><span data-stu-id="abeb9-111">Example</span></span>  
 <span data-ttu-id="abeb9-112">因為類別類型 `U` 是 `struct S` 的成員，但永遠不會進行初始化，所以下列程式碼會產生 CS1060。</span><span class="sxs-lookup"><span data-stu-id="abeb9-112">The following code generates CS1060 because the class type `U` is a member of the `struct S` but is never initialized.</span></span>  
  
```csharp  
// cs1060.cs  
namespace CS1060  
{      
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }    
}  
```
