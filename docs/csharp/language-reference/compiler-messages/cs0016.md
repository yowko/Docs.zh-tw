# Compiler Error CS0016
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

無法寫入輸出檔 'file' \-\- 'reason'  
  
 編譯器無法寫入輸出檔。  請檢查檔案路徑，以確定檔案確實存在。  如果該位置已存在之前建置 \(Build\) 的檔案，請確定可寫入該檔案，且目前沒有處理序鎖定該檔案。  例如，請確定可執行檔未於嘗試建置時執行。
