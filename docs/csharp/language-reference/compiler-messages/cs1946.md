# <a name="compiler-error-cs1946"></a><span data-ttu-id="2fd42-102">編譯器錯誤 CS1946</span><span class="sxs-lookup"><span data-stu-id="2fd42-102">Compiler Error CS1946</span></span>

<span data-ttu-id="2fd42-103">匿名方法運算式無法轉換為運算式樹狀架構。</span><span class="sxs-lookup"><span data-stu-id="2fd42-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
<span data-ttu-id="2fd42-104">匿名方法代表一組陳述式，但運算式樹狀架構不得包含陳述式。</span><span class="sxs-lookup"><span data-stu-id="2fd42-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="2fd42-105">因此，無法以運算式樹狀架構表示匿名方法。</span><span class="sxs-lookup"><span data-stu-id="2fd42-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="2fd42-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="2fd42-106">To correct this error</span></span>

1. <span data-ttu-id="2fd42-107">將匿名方法變更為 Lambda 運算式。</span><span class="sxs-lookup"><span data-stu-id="2fd42-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2fd42-108">範例</span><span class="sxs-lookup"><span data-stu-id="2fd42-108">Example</span></span>

<span data-ttu-id="2fd42-109">下列範例會產生 CS1946：</span><span class="sxs-lookup"><span data-stu-id="2fd42-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
