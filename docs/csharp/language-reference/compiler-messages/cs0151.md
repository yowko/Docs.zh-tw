# <a name="compiler-error-cs0151"></a>編譯器錯誤 CS0151
必須是整數型別的值  
  
 在需要整數資料型別的情況下使用了變數。 如需詳細資訊，請參閱[類型](../../../csharp/programming-guide/types/index.md)。  
  
## <a name="example"></a>範例  
 如果沒有轉換，或是可用的隱含轉換導致模稜兩可的情況，就會發生此錯誤。 下列範例會產生 CS0151。  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```  
  
## <a name="example"></a>範例  
 在 Visual Studio 2008 和更新版本中，[void](../../../csharp/language-reference/keywords/void.md) 方法引動過程會產生 CS0151。 您可以呼叫傳回整數型別 (例如 [int](../../../csharp/language-reference/keywords/int.md) 或 [long](../../../csharp/language-reference/keywords/long.md)) 的方法，來修正此錯誤。  
  
```csharp  
class C  
{  
    static void Main()  
    {  
  
        switch (M()) // CS0151  
        {  
            default:  
                break;  
        }  
    }  
  
    static void M()  
    {  
    }  
}  
```
