# /help, /? (C# Compiler Options)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

這個選項會將編譯器選項清單和每個選項的簡短說明傳送到 stdout。  
  
## 語法  
  
```  
/help  
/?  
```  
  
## 備註  
 如果編譯中包含這個選項，就不會建立輸出檔，而且也不會進行編譯。  
  
 在 Visual Studio 中無法使用這個編譯器選項，而且無法利用程式設計的方式變更它。
