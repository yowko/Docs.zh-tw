# <a name="compiler-error-cs1551"></a>編譯器錯誤 CS1551
索引子至少要有一個參數  
  
 已宣告未採用任何引數的 [索引子](../programming-guide/indexers/index.md) 。  
  
 下列範例會產生 CS1551：  
  
```csharp  
// CS1551.cs  
public class MyClass  
{  
   int intI;  
  
   int this[]   // CS1551  
   // try the following line instead  
   // int this[int i]  
   {  
      get  
      {  
         return intI;  
      }  
      set  
      {  
         intI = value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
