 <span data-ttu-id="2ac81-104">如果部分方法具有實作，泛型類型條件約束必須與方法簽章上所定義的條件約束完全相同。</span><span class="sxs-lookup"><span data-stu-id="2ac81-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
