# <a name="compiler-error-cs0683"></a>編譯器錯誤 CS0683
'explicitmethod' 明確方法實作無法實作 'method'，因為它是存取子。  
  
 下列範例會產生 CS0683：  
  
```csharp  
// CS0683.cs  
interface IExample  
{  
   int Test { get; }  
}  
  
class CExample : IExample  
{  
   int IExample.get_Test() { return 0; } // CS0683  
   int IExample.Test { get{ return 0; } } // correct  
}  
```
