# <a name="compiler-error-cs1714"></a>編譯器錯誤 CS1714
TypeName1 的基底類別或介面無法解析或者無效  
  
 您正在從無法解析 (例如編譯器找不到它) 或不正確的類別或介面實作 TypeName1。 解決方案為判斷是這兩種情況的何者，然後更正確地指定類型的位置，或在基底類別中修正任何編譯器錯誤。
