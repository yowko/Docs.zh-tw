# <a name="compiler-error-cs0210"></a><span data-ttu-id="908f9-102">編譯器錯誤 CS0210</span><span class="sxs-lookup"><span data-stu-id="908f9-102">Compiler Error CS0210</span></span>
<span data-ttu-id="908f9-103">在 fixed 或 using 陳述式宣告中，必須提供初始設定式。</span><span class="sxs-lookup"><span data-stu-id="908f9-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="908f9-104">您必須宣告並初始化 [fixed 陳述式](../language-reference/keywords/fixed-statement.md)中的變數。</span><span class="sxs-lookup"><span data-stu-id="908f9-104">You must declare and initialize the variable in a [fixed statement](../language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="908f9-105">如需詳細資訊，請參閱[不安全的程式碼和指標](../programming-guide/unsafe-code-pointers/index.md)。</span><span class="sxs-lookup"><span data-stu-id="908f9-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="908f9-106">下列範例會產生 CS0210：</span><span class="sxs-lookup"><span data-stu-id="908f9-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="908f9-107">下列範例也會產生 CS0210，因為 [using 陳述式](../language-reference/keywords/using-statement.md) 沒有初始設定式。</span><span class="sxs-lookup"><span data-stu-id="908f9-107">The following sample also generates CS0210 because the [using statement](../language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
