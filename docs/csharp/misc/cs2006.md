# 編譯器錯誤 CS2006
命令列語法錯誤: 遺漏 'option' 選項的 'text'  
  
 *option* 的語法需要其他文字。 如需相關資訊，請參閱[編譯器選項](../../csharp/language-reference/compiler-options/index.md)。
