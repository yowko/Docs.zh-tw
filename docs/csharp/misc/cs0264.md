# <a name="compiler-error-cs0264"></a>編譯器錯誤 CS0264
'type' 的部分宣告必須有相同順序的相同類型參數名稱  
  
 如果您在部分宣告中定義泛型類型，而且類型參數在所有部分宣告中的名稱或順序不一致，則會發生這個錯誤。 若要排除這個錯誤，請檢查每個部分宣告的類型參數，並確定使用參數的相同名稱和順序。 如需詳細資訊，請參閱 <<c0> [ 部分類別和方法](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md)還有[泛型型別參數](../../csharp/programming-guide/generics/generic-type-parameters.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0264。  
  
```csharp  
// CS0264.cs  
  
partial class MyClass<T>  // CS0264  
{  
}  
  
partial class MyClass <MyType>  
{  
}  
```
