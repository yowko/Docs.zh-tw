# <a name="compiler-error-cs0513"></a>編譯器錯誤 CS0513
'function' 為抽象方法，但包含於非抽象類別 'class' 之中  
  
 方法不能是非抽象類別的 [抽象](../language-reference/keywords/abstract.md) 成員。  
  
 下列範例會產生 CS0513：  
  
```csharp  
// CS0513.cs  
namespace x  
{  
   public class clx  
   {  
      abstract public void f();   // CS0513, abstract member of nonabstract class  
   }  
}  
```
