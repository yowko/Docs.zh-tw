# <a name="compiler-error-cs0572"></a><span data-ttu-id="52535-102">編譯器錯誤 CS0572</span><span class="sxs-lookup"><span data-stu-id="52535-102">Compiler Error CS0572</span></span>
<span data-ttu-id="52535-103">'type': 不可透過運算式參考類型; 請嘗試改用 'path_to_type'</span><span class="sxs-lookup"><span data-stu-id="52535-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="52535-104">已嘗試透過識別項存取類別的成員，在此情況下不允許這麼做。</span><span class="sxs-lookup"><span data-stu-id="52535-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="52535-105">下列範例會產生 CS0572：</span><span class="sxs-lookup"><span data-stu-id="52535-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
