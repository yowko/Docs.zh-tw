# <a name="compiler-error-cs2017"></a>編譯器錯誤 CS2017
在建置模組或程式庫時不能指定 /main  
  
 當您建置 [/target:library](../language-reference/compiler-options/target-library-compiler-option.md)時無法指定主要進入點。  
  
 下列範例會產生 CS2017：  
  
```csharp  
// CS2017.cs  
// compile with: /main:MyClass /target:library  
// CS2017 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
