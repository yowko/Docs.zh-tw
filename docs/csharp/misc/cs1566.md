# <a name="compiler-error-cs1566"></a>編譯器錯誤 CS1566
讀取資源檔 'file' 時發生錯誤 — 'reason'  
  
 編譯器無法將檔名傳遞給 [/resource](../../csharp/language-reference/compiler-options/resource-compiler-option.md) 編譯器選項。
