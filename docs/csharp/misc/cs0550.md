## <a name="example"></a><span data-ttu-id="58274-107">範例</span><span class="sxs-lookup"><span data-stu-id="58274-107">Example</span></span>  

 <span data-ttu-id="58274-108">下列範例會產生 CS0550：</span><span class="sxs-lookup"><span data-stu-id="58274-108">The following sample generates CS0550.</span></span>  
  
```csharp  
// CS0550.cs  
namespace x  
{  
   interface ii  
   {  
      int i  
      {  
         get;  
         // add the following accessor to resolve this CS0550  
         // set;  
      }  
   }  
  
   public class a : ii  
   {  
      int ii.i  
      {  
         get  
         {  
            return 0;  
         }  
         set {}   // CS0550  no set in interface  
      }  
  
      public static void Main() {}  
   }  
}  
```
