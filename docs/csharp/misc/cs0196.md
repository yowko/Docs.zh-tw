# 編譯器錯誤 CS0196
只能使用一個值對指標編製索引  
  
 指標不能有多個索引。 如需詳細資訊，請參閱[指標類型](../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md)。  
  
 下列範例會產生 CS0196：  
  
```  
// CS0196.cs public class MyClass { public static void Main () { int *i = null; int j = 0; j = i[1,2];   // CS0196 // try the following line instead // j = i[1]; } }  
```
