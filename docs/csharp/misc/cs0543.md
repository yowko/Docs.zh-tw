# 編譯器錯誤 CS0543
'enumeration' : 就其類型而言，此列舉值過大  
  
 指派給[列舉](../../csharp/language-reference/keywords/enum.md)中某個項目的值超過資料類型的範圍。  
  
 下列範例會產生 CS0543：  
  
```  
// CS0543.cs namespace x { enum I : byte {a = 255, b, c}   // CS0543 public class clx { public static int Main() { return 0; } } }  
```
