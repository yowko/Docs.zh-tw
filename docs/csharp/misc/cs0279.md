## <a name="example"></a><span data-ttu-id="b5705-109">範例</span><span class="sxs-lookup"><span data-stu-id="b5705-109">Example</span></span>  
 <span data-ttu-id="b5705-110">下列範例會產生 CS0279：</span><span class="sxs-lookup"><span data-stu-id="b5705-110">The following example generates CS0279:</span></span>

public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;

    }
