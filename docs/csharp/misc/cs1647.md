# 編譯器錯誤 CS1647
'code' 附近的運算式太長或太複雜，造成編譯困難  
  
 處理您的程式碼時，編譯器中發生堆疊溢位。 若要解決這個錯誤，請簡化您的程式碼。 如果您的程式碼有效，請連絡[產品支援](http://msdn.microsoft.com/zh-tw/77e75b8b-817d-45bf-9c38-458930d873b4)。
