# <a name="compiler-error-cs1719"></a>編譯器錯誤 CS1719
讀取 Win32 資源檔 'File Name' 時發生錯誤 -- 'reason'  
  
 嘗試讀取 Win32 資源檔失敗，因為發生錯誤訊息中所指定的原因，通常類似「找不到檔案」或「拒絕存取」。 若要解決這個錯誤，請更正原因所描述的問題。
