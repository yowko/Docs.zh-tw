# 編譯器錯誤 CS1011
空的字元常值  
  
 [char](../../csharp/language-reference/keywords/char.md) 已宣告並初始化為 null。`char` 的初始設定必須指定一個字元。  
  
 下列範例會產生 CS1011：  
  
```  
// CS1011.cs class Sample { public char CharField = '';   // CS1011 }  
```
