# <a name="compiler-error-cs0212"></a><span data-ttu-id="6de8e-102">編譯器錯誤 CS0212</span><span class="sxs-lookup"><span data-stu-id="6de8e-102">Compiler Error CS0212</span></span>
<span data-ttu-id="6de8e-103">您只能取得 fixed 陳述式初始設定式中 unfixed 運算式的位址</span><span class="sxs-lookup"><span data-stu-id="6de8e-103">You can only take the address of an unfixed expression inside of a fixed statement initializer</span></span>  
  
 <span data-ttu-id="6de8e-104">如需詳細資訊，請參閱[不安全的程式碼和指標](../../csharp/programming-guide/unsafe-code-pointers/index.md)。</span><span class="sxs-lookup"><span data-stu-id="6de8e-104">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="6de8e-105">下列範例會示範如何取得 unfixed 運算式的位址。</span><span class="sxs-lookup"><span data-stu-id="6de8e-105">The following sample shows how to take the address of an unfixed expression.</span></span> <span data-ttu-id="6de8e-106">下列範例會產生 CS0212。</span><span class="sxs-lookup"><span data-stu-id="6de8e-106">The following sample generates CS0212.</span></span>  
  
```  
// CS0212a.cs  
// compile with: /unsafe /target:library  
  
public class A {  
   public int iField = 5;  
  
   unsafe public void M() {   
      A a = new A();  
      int* ptr = &a.iField;   // CS0212   
   }  
  
   // OK  
   unsafe public void M2() {  
      A a = new A();  
      fixed (int* ptr = &a.iField) {}  
   }  
}  
```  
  
 <span data-ttu-id="6de8e-107">下列範例也會產生 CS0212，並說明如何解決錯誤：</span><span class="sxs-lookup"><span data-stu-id="6de8e-107">The following sample also generates CS0212 and shows how to resolve the error:</span></span>  
  
```  
// CS0212b.cs  
// compile with: /unsafe /target:library  
using System;  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      // Null-terminated ASCII characters in an sbyte array   
      sbyte[] sbArr1 = new sbyte[] { 0x41, 0x42, 0x43, 0x00 };  
      sbyte* pAsciiUpper = &sbArr1[0];   // CS0212  
      // To resolve this error, delete the previous line and   
      // uncomment the following code:  
      // fixed (sbyte* pAsciiUpper = sbArr1)  
      // {  
      //    String szAsciiUpper = new String(pAsciiUpper);  
      // }  
   }  
}  
```
