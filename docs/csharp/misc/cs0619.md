# <a name="compiler-error-cs0619"></a><span data-ttu-id="53af9-102">編譯器錯誤 CS0619</span><span class="sxs-lookup"><span data-stu-id="53af9-102">Compiler Error CS0619</span></span>
<span data-ttu-id="53af9-103">'member' 已過時: 'text'</span><span class="sxs-lookup"><span data-stu-id="53af9-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="53af9-104">類別成員被標記為[過時](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488)屬性，因此，當參考類別成員時，會發出錯誤。</span><span class="sxs-lookup"><span data-stu-id="53af9-104">A class member was marked with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="53af9-105">下列範例會產生 CS0619：</span><span class="sxs-lookup"><span data-stu-id="53af9-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
