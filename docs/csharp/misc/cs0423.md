<span data-ttu-id="2094e-103">由於 'class' 具有 ComImport 屬性，'method' 必須為 extern 或 abstract</span><span class="sxs-lookup"><span data-stu-id="2094e-103">Since 'class' has the ComImport attribute, 'method' must be extern or abstract</span></span>  
  
 <span data-ttu-id="2094e-104">指定 ComImport 屬性表示是要從 COM 模組匯入類別的實作。</span><span class="sxs-lookup"><span data-stu-id="2094e-104">Specifying the ComImport attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="2094e-105">可能不會定義額外的方法。</span><span class="sxs-lookup"><span data-stu-id="2094e-105">Additional methods may not be defined.</span></span>

```csharp  
// CS0423.cs

[  
  ComImport,  
  Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829")

  {  
    ImageProperties i = new ImageProperties();  
  }  
}  
```
