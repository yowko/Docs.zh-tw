<span data-ttu-id="45a1a-103">'method' 沒有任何多載符合委派 'delegate'</span><span class="sxs-lookup"><span data-stu-id="45a1a-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="45a1a-104">建立委派的嘗試失敗，因為未使用正確的特徵標記。</span><span class="sxs-lookup"><span data-stu-id="45a1a-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="45a1a-105">委派的執行個體必須使用和委派宣告一樣的特徵標記宣告。</span><span class="sxs-lookup"><span data-stu-id="45a1a-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="45a1a-106">您可以調整方法或委派特徵標記來解決這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="45a1a-106">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="45a1a-107">如需詳細資訊，請參閱[委派](../programming-guide/delegates/index.md)。</span><span class="sxs-lookup"><span data-stu-id="45a1a-107">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="45a1a-108">下列範例會產生 CS0123。</span><span class="sxs-lookup"><span data-stu-id="45a1a-108">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
