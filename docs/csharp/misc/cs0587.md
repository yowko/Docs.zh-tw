# <a name="compiler-error-cs0587"></a>編譯器錯誤 CS0587

編譯器內部錯誤：階段 'stage'

 請嘗試判斷編譯器失敗的原因是否為無法剖析未預期的語法。 如果不是這種情況，請嘗試[與我們交談](/visualstudio/ide/feedback-options)。
 
> [!NOTE]
> Roslyn 中已不再使用此編譯器錯誤。
