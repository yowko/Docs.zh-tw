# 編譯器錯誤 CS1023
內嵌的陳述式不能為宣告或標記陳述式  
  
 內嵌陳述式 \(例如 **if** 陳述式後面的陳述式\) 不可以包含宣告或標籤陳述式。  
  
 下列範例會產生 CS1023 兩次：  
  
```  
// CS1023.cs public class a { public static void Main() { if (1) int i;      // CS1023, declaration is not valid here if (1) xx : i++;   // CS1023, labeled statement is not valid here } }  
```
