# 編譯器警告 (層級 1) CS0602
'old\_feature' 功能已被取代。 請改用 'new\_feature'  
  
 仍然支援在程式碼中使用的語言功能 \(*old\_feature*\)，但該支援可能會在未來版本中移除。 您應該改用建議的語法 \(*new\_feature*\)。
