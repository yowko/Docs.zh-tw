# <a name="compiler-error-cs1552"></a>編譯器錯誤 CS1552
陣列類型規範 [] 必須出現在參數名稱之前  
  
 陣列類型規範的位置在陣列宣告的變數名稱後面。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1552：  
  
```  
// CS1552.cs  
public class C  
{  
    public static void Main(string args[])   // CS1552  
    // try the following line instead  
    // public static void Main(string [] args)  
    {  
    }  
}  
```
