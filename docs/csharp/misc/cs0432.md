# <a name="compiler-error-cs0432"></a><span data-ttu-id="1189f-102">編譯器錯誤 CS0432</span><span class="sxs-lookup"><span data-stu-id="1189f-102">Compiler Error CS0432</span></span>
<span data-ttu-id="1189f-103">找不到別名 'identifier'</span><span class="sxs-lookup"><span data-stu-id="1189f-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="1189f-104">當您在非別名的識別項右側使用 "::" 時，會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="1189f-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="1189f-105">若要解決這個錯誤，請改用 "."。</span><span class="sxs-lookup"><span data-stu-id="1189f-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="1189f-106">下列範例會產生 CS0432：</span><span class="sxs-lookup"><span data-stu-id="1189f-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
