# <a name="compiler-error-cs0670"></a>編譯器錯誤 CS0670
欄位不能有 void 類型  
  
 欄位已宣告為類型 [void](../../csharp/language-reference/keywords/void.md)。  
  
 下列範例會產生 CS0670：  
  
```csharp  
// CS0670.cs  
class C  
{  
   void f;   // CS0670  
   // try the following line instead  
   // public int f;  
  
   public static void Main()  
   {  
      C myc = new C();  
      myc.f = 0;  
   }  
}  
```
