# 編譯器錯誤 CS0247
stackalloc 無法使用負值大小  
  
 負值已傳遞至 [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) 陳述式。  
  
 下列範例會產生 CS0247：  
  
```  
// CS0247.cs // compile with: /unsafe public class MyClass { unsafe public static void Main() { int *p = stackalloc int [-30];   // CS0247 } }  
```
