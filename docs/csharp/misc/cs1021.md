# 編譯器錯誤 CS1021
整數常數太大  
  
 指派給[整數類型](../../csharp/language-reference/keywords/integral-types-table.md)的值大於可能不帶正負號的最大整數值。  
  
 下列範例會產生 CS1021：  
  
```  
// CS1021.cs enum F : int { a=(int)2590000000000000000000   // CS1021 } public class clx { public static void Main() { } }  
```
