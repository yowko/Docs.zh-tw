# 編譯器錯誤 CS0132
'constructor' : 靜態的建構函式不能使用參數  
  
 不能以一個或多個參數宣告[靜態](../../visual-basic/language-reference/modifiers/static.md)建構函式。 如需詳細資訊，請參閱[建構函式](../../csharp/programming-guide/classes-and-structs/constructors.md)。  
  
 下列範例會產生 CS0132：  
  
```  
// CS0132.cs namespace MyNamespace { public class MyClass { public MyClass(int i) { } } public class MyClass2 : MyClass { static MyClass2(int i)   // CS0132 { } } }  
```
