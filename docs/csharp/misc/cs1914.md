 <span data-ttu-id="c60b7-104">根據定義的物件初始設定式，會初始化物件或類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="c60b7-104">Object initializers by definition initialize objects, or instances, of classes.</span></span> <span data-ttu-id="c60b7-105">它們無法用來初始化類型的 `static` 欄位。</span><span class="sxs-lookup"><span data-stu-id="c60b7-105">They cannot be used to initialize a `static` field of a type.</span></span> <span data-ttu-id="c60b7-106">不論建立多少個類別執行個體，只會有一份 `static` 欄位。</span><span class="sxs-lookup"><span data-stu-id="c60b7-106">No matter how many instances of a class are created, there is only one copy of a `static` field.</span></span>

## <a name="example"></a><span data-ttu-id="c60b7-109">範例</span><span class="sxs-lookup"><span data-stu-id="c60b7-109">Example</span></span>  
 <span data-ttu-id="c60b7-110">下列程式碼會產生 CS1914，因為初始設定式嘗試初始化 `TestClass.Number` 欄位，它是 `static`：</span><span class="sxs-lookup"><span data-stu-id="c60b7-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }      
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
