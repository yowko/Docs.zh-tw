# 編譯器錯誤 CS0708
'field': 不能在靜態類別中宣告執行個體成員  
  
 如果您在已宣告靜態的類別中宣告非靜態的成員，就會發生此錯誤。 不可能建立靜態類別的執行個體，因此執行個體變數沒有意義。**static** 關鍵字應該套用至靜態類別的所有成員。  
  
 下列範例會產生 CS0708：  
  
```  
// CS0708.cs // compile with: /target:library public static class C { int i;  // CS0708 static int j;  // OK }  
```
