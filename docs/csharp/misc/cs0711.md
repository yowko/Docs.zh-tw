# 編譯器錯誤 CS0711
靜態類別不能包含解構函式  
  
 靜態類別無法具現化，因此不需要建構函式或解構函式。 若要避免這個錯誤，請移除靜態類別中的所有解構函式，或如果您真的想要建構並破壞執行個體，請將類別變成非靜態。  
  
 下列範例會產生 CS0711：  
  
```  
// CS0711.cs public static class C { ~C()  // CS0711 { } public static void Main() { } }  
```
