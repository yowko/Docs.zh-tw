# 編譯器警告 (層級 1) CS3006
只有 ref\/out 或陣列陣序差異的多載方法 'method'，不符合 CLS 標準  
  
 方法無法根據 [ref](../../csharp/language-reference/keywords/ref.md) 或 [out](../../csharp/language-reference/keywords/out.md) 參數多載，而仍然符合 Common Language Specification \(CLS\)。 如需 CLS 符合性的詳細資訊，請參閱[撰寫符合 CLS 標準的程式碼](http://msdn.microsoft.com/zh-tw/4c705105-69a2-4e5e-b24e-0633bc32c7f3)和[語言獨立性以及與語言無關的元件](../Topic/Language%20Independence%20and%20Language-Independent%20Components.md)。  
  
## 範例  
 下列範例會產生 CS3006： 若要解決這個警告，請將組件層級屬性標記為註解或移除其中一個方法定義。  
  
```  
// CS3006.cs using System; [assembly: CLSCompliant(true)] public class MyClass { public void f(int i) { } public void f(ref int i)   // CS3006 { } public static void Main() { } }  
```
