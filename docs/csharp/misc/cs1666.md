# <a name="compiler-error-cs1666"></a><span data-ttu-id="8795e-102">編譯器錯誤 CS1666</span><span class="sxs-lookup"><span data-stu-id="8795e-102">Compiler Error CS1666</span></span>
<span data-ttu-id="8795e-103">您不能使用包含在 unfixed 運算式中的固定大小緩衝區。</span><span class="sxs-lookup"><span data-stu-id="8795e-103">You cannot use fixed size buffers contained in unfixed expressions.</span></span> <span data-ttu-id="8795e-104">請嘗試使用 fixed 陳述式。</span><span class="sxs-lookup"><span data-stu-id="8795e-104">Try using the fixed statement.</span></span>  
  
 <span data-ttu-id="8795e-105">如果您在包含本身不固定之類別的運算式中使用固定大小的緩衝區，就會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="8795e-105">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="8795e-106">執行階段可以自由地四處移動非固定的類別，以最佳化記憶體存取，這可能導致使用固定大小緩衝區時發生錯誤。</span><span class="sxs-lookup"><span data-stu-id="8795e-106">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="8795e-107">若要避免這個錯誤，請在陳述式上使用 `fixed` 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="8795e-107">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8795e-108">範例</span><span class="sxs-lookup"><span data-stu-id="8795e-108">Example</span></span>  
 <span data-ttu-id="8795e-109">下列範例會產生 CS1666：</span><span class="sxs-lookup"><span data-stu-id="8795e-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);   
   }   
}  
```
