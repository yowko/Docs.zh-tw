# <a name="compiler-error-cs0238"></a><span data-ttu-id="77a09-102">編譯器錯誤 CS0238</span><span class="sxs-lookup"><span data-stu-id="77a09-102">Compiler Error CS0238</span></span>
<span data-ttu-id="77a09-103">'member' 不是 override，因此無法成為密封 (sealed)</span><span class="sxs-lookup"><span data-stu-id="77a09-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="77a09-104">[sealed](../../csharp/language-reference/keywords/sealed.md) 已用於也未以 [override](../../csharp/language-reference/keywords/override.md)標記的成員。</span><span class="sxs-lookup"><span data-stu-id="77a09-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="77a09-105">如需詳細資訊，請參閱[繼承](../../csharp/programming-guide/classes-and-structs/inheritance.md)。</span><span class="sxs-lookup"><span data-stu-id="77a09-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="77a09-106">下列範例會產生 CS0238：</span><span class="sxs-lookup"><span data-stu-id="77a09-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
