# <a name="compiler-error-cs0719"></a>編譯器錯誤 CS0719
'type': 陣列項目不得為靜態類型  
  
 靜態類型的陣列毫無意義，因為陣列項目是執行個體，但它不可能建立靜態類型的執行個體。  
  
 下列範例會產生 CS0719：  
  
```  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
