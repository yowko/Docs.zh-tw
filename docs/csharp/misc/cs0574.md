# <a name="compiler-error-cs0574"></a>編譯器錯誤 CS0574
解構函式的名稱必須符合類別的名稱  
  
 解構函式名稱的前面必須是加上波狀符號 (~) 的類別名稱。  
  
 下列範例會產生 CS0574：  
  
```csharp  
// CS0574.cs  
namespace x  
{  
   public class iii  
   {  
      ~iiii()   // CS0574  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
