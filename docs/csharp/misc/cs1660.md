# <a name="compiler-error-cs1660"></a><span data-ttu-id="da8d3-102">編譯器錯誤 CS1660</span><span class="sxs-lookup"><span data-stu-id="da8d3-102">Compiler Error CS1660</span></span>
<span data-ttu-id="da8d3-103">無法將匿名方法區塊轉換成類型 'type'，因為它不是委派類型</span><span class="sxs-lookup"><span data-stu-id="da8d3-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="da8d3-104">如果您嘗試將匿名方法區塊指派或轉換為不是委派類型的類型，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="da8d3-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="da8d3-105">下列範例會產生 CS1660：</span><span class="sxs-lookup"><span data-stu-id="da8d3-105">The following sample generates CS1660:</span></span>  
  
```  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
