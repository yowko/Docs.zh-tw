# <a name="compiler-error-cs0026"></a>編譯器錯誤 CS0026
關鍵字 'this' 在靜態屬性、靜態方法或靜態欄位初始設定式中無效。  
  
 [this](../../csharp/language-reference/keywords/this.md) 關鍵字參考的物件是某個類型的執行個體。 由於靜態方法與任何包含類別執行個體皆無關，因此 "this" 關鍵字是無意義且不被允許的。 如需詳細資訊，請參閱 <<c0> [ 靜態類別和靜態類別成員](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md)並[物件](../../csharp/programming-guide/classes-and-structs/objects.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0026：  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;     
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
