# <a name="compiler-error-cs0441"></a>編譯器錯誤 CS0441
'class': 類別不可同時為 static 和 sealed  
  
 當您宣告同時為 static 和 sealed 的類別時，會發生這個錯誤。 static 類別原本就進行密封，因此不需要 sealed 修飾詞。 若要解決，請僅使用一個修飾詞。  
  
 下列範例會產生 CS0441：  
  
```csharp  
// CS0441.cs  
sealed static class MyClass { }   // CS0441  
```
