# <a name="compiler-error-cs0037"></a>編譯器錯誤 CS0037
無法將 null 轉換成 'type'，因為它是不可為 Null 的實值類型  
  
 編譯器無法將 `null` 指派給實數值型別;`null` 只能指派給[參考型別](../language-reference/keywords/reference-types.md)或可為 null 的實[數值型別](../language-reference/builtin-types/nullable-value-types.md)。 [結構](../language-reference/builtin-types/struct.md) 類型是實值類型。
  
 下列範例會產生 CS0037：  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
