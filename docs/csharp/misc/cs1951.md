# <a name="compiler-error-cs1951"></a><span data-ttu-id="38ed1-102">編譯器錯誤 CS1951</span><span class="sxs-lookup"><span data-stu-id="38ed1-102">Compiler Error CS1951</span></span>

 <span data-ttu-id="38ed1-104">運算式樹狀結構只會將運算式表示為資料結構。</span><span class="sxs-lookup"><span data-stu-id="38ed1-104">An expression tree just represents expressions as data structures.</span></span> <span data-ttu-id="38ed1-105">沒有任何方法可將特定記憶體位置表示為透過傳址方式傳遞參數時的必要記憶體位置。</span><span class="sxs-lookup"><span data-stu-id="38ed1-105">There is no way to represent specific memory locations as is required when you pass a parameter by reference.</span></span>

## <a name="example"></a><span data-ttu-id="38ed1-108">範例</span><span class="sxs-lookup"><span data-stu-id="38ed1-108">Example</span></span>  
 <span data-ttu-id="38ed1-109">下列範例會產生 CS1951：</span><span class="sxs-lookup"><span data-stu-id="38ed1-109">The following example generates CS1951:</span></span>
