# 編譯器錯誤 CS0263
'type' 的部分宣告不可指定不同的基底類別  
  
 在部分宣告中定義類型時，請在每個部分宣告中指定相同的基底類型。 如需詳細資訊，請參閱[部分類別和方法](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md)。  
  
 下列範例會產生 CS0263：  
  
```  
  
// CS0263.cs // compile with: /target:library class B1 { } class B2 { } partial class C : B1  // CS0263 – is the base class B1 or B2? { } partial class C : B2 { }  
```
