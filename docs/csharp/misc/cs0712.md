# <a name="compiler-error-cs0712"></a>編譯器錯誤 CS0712
無法建立靜態類別 'static class' 的執行個體  
  
 您不可以建立靜態類別的執行個體。 靜態類別設計成包含靜態欄位和方法，但可能未進行具現化。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0712：  
  
```  
// CS0712.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
    public static void Main()  
    {  
        SC sc = new SC();  // CS0712  
    }  
}  
```
