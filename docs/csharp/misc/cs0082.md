# <a name="compiler-error-cs0082"></a>編譯器錯誤 CS0082
類型 'type' 已保留具有相同參數類型名為 'name' 的成員  
  
 屬性在編譯期間會轉譯成識別項前面有 `get_` 及/或 `set_` 的方法。 如果定義了您自己與方法名稱發生衝突的方法，就會發生錯誤。  
  
## <a name="example"></a>範例  
 下例會產生 CS0082：  
  
```csharp  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
