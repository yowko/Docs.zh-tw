# <a name="compiler-error-cs0255"></a>編譯器錯誤 CS0255
在 catch 或 finally 區塊中不可使用 stackalloc  
  
您無法使用[stackalloc 運算子](../language-reference/operators/stackalloc.md)中[攔截](../language-reference/keywords/try-catch.md)或是[最後](../language-reference/keywords/try-catch-finally.md)區塊。 如需詳細資訊，請參閱 <<c0> [ 例外狀況和例外狀況處理](../programming-guide/exceptions/index.md)。  
  
下列範例會產生 CS0255：  
  
```csharp  
// CS0255.cs  
// compile with: /unsafe  
using System;  
  
public class TestTryFinally  
{  
   public static unsafe void Test()  
   {  
      int i = 123;  
      string s = "Some string";  
      object o = s;  
  
      try  
      {  
         // Conversion is not valid; o contains a string not an int  
         i = (int) o;  
      }  
      finally  
      {  
         Console.Write("i = {0}", i);  
         int* fib = stackalloc int[100];   // CS0255  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
