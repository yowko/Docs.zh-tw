# <a name="compiler-error-cs0594"></a>編譯器錯誤 CS0594
浮點常數的值超出類型 'type' 的範圍  
  
 已將值指派給對這個資料類型的變數而言太大的浮點變數。 如需資料類型中所允許之值範圍的詳細資訊，請參閱 [整數類型表](../../csharp/language-reference/builtin-types/integral-numeric-types.md) 。  
  
 下列範例會產生 CS0594：  
  
```csharp  
// CS0594.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         float f = 6.77777777777E400;   // CS0594, value too large  
      }  
   }  
}  
```
