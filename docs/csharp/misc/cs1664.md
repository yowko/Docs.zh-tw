# <a name="compiler-error-cs1664"></a>編譯器錯誤 CS1664
長度為 'length' 且類型為 'type' 的固定大小緩衝區太大  
  
 固定大小緩衝區大小上限 (透過長度乘上項目大小所決定) 是 2^31 = 268435455。
