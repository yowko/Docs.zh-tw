# 編譯器錯誤 CS1586
建立陣列必須有陣列大小或陣列初始設定式  
  
 陣列宣告不正確。  
  
 下列範例會產生 CS1586：  
  
```  
// CS1586.cs using System; class MyClass { public static void Main() { int[] a = new int[];   // CS1586 // try the following line instead int[] b = new int[5]; } }  
```
