# <a name="compiler-warning-level-2-cs0253"></a>編譯器警告 (層級 2) CS0253
可能誤用了參考比較; 若要進行數值比較，請將右側轉型為類型 'type'  
  
 編譯器正在進行參考比較。 如果您想要比較字串值，請將運算式的右邊轉換為 `type`。  
  
 下列範例會產生 CS0253：  
  
```csharp 
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
