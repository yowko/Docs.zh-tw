## <a name="example"></a><span data-ttu-id="b1f85-110">範例</span><span class="sxs-lookup"><span data-stu-id="b1f85-110">Example</span></span>  
 <span data-ttu-id="b1f85-111">下列程式碼會產生 CS0736，因為 `Program.testMethod` 宣告為靜態：</span><span class="sxs-lookup"><span data-stu-id="b1f85-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
