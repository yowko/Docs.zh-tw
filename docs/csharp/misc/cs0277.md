## <a name="example"></a><span data-ttu-id="cbe65-109">範例</span><span class="sxs-lookup"><span data-stu-id="cbe65-109">Example</span></span>  
 <span data-ttu-id="cbe65-110">下列範例會產生 CS0277：</span><span class="sxs-lookup"><span data-stu-id="cbe65-110">The following example generates CS0277:</span></span>  
  
```csharp

    int Property  
    {  
        get;  
        set;  
    }

{  
    public int Property  
    {  
        get { return 0; }  
        // Try this instead:  
        //set { }  
        protected set { }  
    }  
}  
```
