# 編譯器錯誤 CS0575
只有類別類型可以包含解構函式  
  
 [結構](../../csharp/language-reference/keywords/struct.md)不能包含解構函式。  
  
 下列範例會產生 CS0575：  
  
```  
// CS0575.cs namespace x { public struct iii { ~iii()   // CS0575 { } public static void Main() { } } }  
```
