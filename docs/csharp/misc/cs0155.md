# <a name="compiler-error-cs0155"></a><span data-ttu-id="3fc4f-102">編譯器錯誤 CS0155</span><span class="sxs-lookup"><span data-stu-id="3fc4f-102">Compiler Error CS0155</span></span>
<span data-ttu-id="3fc4f-103">類型 catch 或 throw 必須衍生自 System.Exception</span><span class="sxs-lookup"><span data-stu-id="3fc4f-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="3fc4f-104">嘗試將未衍生自 **System.Exception** 的資料類型傳遞至 [catch](../language-reference/keywords/try-catch.md) 區塊。</span><span class="sxs-lookup"><span data-stu-id="3fc4f-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="3fc4f-105">只有衍生自 **System.Exception** 的資料類型才能傳遞至 **catch** 區塊。</span><span class="sxs-lookup"><span data-stu-id="3fc4f-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="3fc4f-106">如需詳細資訊, 請參閱[例外狀況和例外狀況處理](../programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="3fc4f-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="3fc4f-107">下列範例會產生 CS0155：</span><span class="sxs-lookup"><span data-stu-id="3fc4f-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
