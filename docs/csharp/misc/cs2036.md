# 編譯器錯誤 CS2036
\/pdb 選項需要同時使用 \/debug 選項。  
  
 只會針對偵錯組建產生程式資料庫檔案。 因此，**\/pdb** 選項在零售組建中沒有意義。  
  
### 更正這個錯誤  
  
-   加入 **\/debug** 編譯器選項。  
  
-   移除 **\/pdb** 編譯器選項。  
  
## 範例  
 下列範例使用 **\/pdb** 選項但未使用 \/debug 選項進行編譯時，會產生 CS2036。  
  
```  
// cs2036.cs // Compile with: /pdb // CS2036 class Test { public static int Main() { return 1; } }  
```
