<span data-ttu-id="0541d-103">在此內容中使用 null 無效</span><span class="sxs-lookup"><span data-stu-id="0541d-103">Use of null is not valid in this context</span></span>  
  
 <span data-ttu-id="0541d-104">下列範例會產生 CS0186：</span><span class="sxs-lookup"><span data-stu-id="0541d-104">The following sample generates CS0186:</span></span>  
  
```csharp  
// CS0186.cs  
using System;  
using System.Collections;  
  
class MyClass   
{  
   static void Main()   
   {  
      // Each of the following lines generates CS0186:  
      foreach (int i in null) {}   // CS0186  
      foreach (int i in (IEnumerable) null) { };   // CS0186  
   }  
}  
```
