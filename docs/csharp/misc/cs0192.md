# 編譯器錯誤 CS0192
不能傳遞靜態唯讀欄位 'name' 的欄位給 ref 或 out \(除非在靜態建構函式中\)  
  
 標上 [readonly](../../visual-basic/language-reference/modifiers/readonly.md) 關鍵字的欄位 \(變數\) 不能傳遞給 [ref](../../csharp/language-reference/keywords/ref.md) 或 [out](../../csharp/language-reference/keywords/out.md) 參數 \(但在建構函式內除外\)。 如需詳細資訊，請參閱[欄位](../../csharp/programming-guide/classes-and-structs/fields.md)。  
  
 如果 `readonly` 欄位是 [static](../../visual-basic/language-reference/modifiers/static.md) 而建構函式未標示為 `static`，則也會產生 CS0192。  
  
## 範例  
 下列範例會產生 CS0192。  
  
```  
// CS0192.cs class MyClass { public readonly int TestInt = 6; static void TestMethod(ref int testInt) { testInt = 0; } MyClass() { TestMethod(ref TestInt);   // OK } public void PassReadOnlyRef() { TestMethod(ref TestInt);   // CS0192 } public static void Main() { } }  
```
