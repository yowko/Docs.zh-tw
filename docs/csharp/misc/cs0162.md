# <a name="compiler-warning-level-2-cs0162"></a>編譯器警告 (層級 2) CS0162

偵測到不會執行到的程式碼
  
編譯器偵測到從未執行的程式碼。

## <a name="example"></a>範例

下列範例會產生 CS0162：

```csharp
// CS0162.cs
// compile with: /W:2
public class Program
{
    public static void Main()
    {
        goto lab1;
        {
            // The following statements cannot be reached:
            int i = 9;   // CS0162
            i++;
        }
    lab1:
        {
        }
    }
}
```
