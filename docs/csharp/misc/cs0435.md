# <a name="compiler-warning-level-2-cs0435"></a><span data-ttu-id="6861c-102">編譯器警告 (層級 2) CS0435</span><span class="sxs-lookup"><span data-stu-id="6861c-102">Compiler Warning (level 2) CS0435</span></span>

 <span data-ttu-id="6861c-105">原始程式檔 (file_2) 中的命名空間與 file_1 中匯入的類型衝突時，會發出這個警告。</span><span class="sxs-lookup"><span data-stu-id="6861c-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="6861c-106">編譯器會使用原始程式檔中的類型。</span><span class="sxs-lookup"><span data-stu-id="6861c-106">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="6861c-107">下列範例會產生 CS0435：</span><span class="sxs-lookup"><span data-stu-id="6861c-107">The following example generates CS0435:</span></span>

```csharp

// compile with: /t:library

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="6861c-109">然後，編譯這個檔案：</span><span class="sxs-lookup"><span data-stu-id="6861c-109">Then, compile this file:</span></span>  
  
```csharp

// compile with: /r:CS0435_1.dll  
  
using System;  
  
namespace Util   
{  
   public class A { }  
}  
  
public class Test   
{  
   public static void Main()   
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
