# <a name="compiler-error-cs1035"></a>編譯器錯誤 CS1035
找到檔案結尾，必須是 '*/'  
  
 開頭與結尾註解分隔符號不成對。  
  
 下列範例會產生 CS1035：  
  
```csharp  
// CS1035.cs  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
/*   // CS1035, needs closing comment  
```
