 <span data-ttu-id="25b50-105">您已搭配使用 "::" 與參考類型的別名。</span><span class="sxs-lookup"><span data-stu-id="25b50-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="25b50-106">若要解決這個錯誤，請使用 "." 運算子。</span><span class="sxs-lookup"><span data-stu-id="25b50-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="25b50-107">下列範例會產生 CS0431：</span><span class="sxs-lookup"><span data-stu-id="25b50-107">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer   
{  
   public class Inner   
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
