# <a name="compiler-error-cs0306"></a>編譯器錯誤 CS0306
類型 'type' 不能用來作為類型引數  
  
 此類型不可當做類型參數來使用。 這可能是因為此類型為指標類型。  
  
 下列範例會產生 CS0306：  
  
```csharp  
// CS0306.cs  
class C<T>  
{  
}  
  
class M  
{  
    // CS0306 – int* not allowed as a type parameter  
     C<int*> f;  
}  
```
