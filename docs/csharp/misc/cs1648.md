 <span data-ttu-id="bccb4-104">當您嘗試修改不允許修改的唯讀欄位成員時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="bccb4-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="bccb4-105">若要解決這個錯誤，請將唯讀欄位的指派限制為建構函式或變數初始設定式，或移除欄位宣告的唯讀關鍵字。</span><span class="sxs-lookup"><span data-stu-id="bccb4-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>

## <a name="example"></a><span data-ttu-id="bccb4-106">範例</span><span class="sxs-lookup"><span data-stu-id="bccb4-106">Example</span></span>

 <span data-ttu-id="bccb4-107">下列範例會產生 CS1648：</span><span class="sxs-lookup"><span data-stu-id="bccb4-107">The following sample generates CS1648:</span></span>

```csharp
// CS1648.cs
public struct Inner
{
    public int i;
}

class Outer
{
    public readonly Inner inner = new Inner();
}

class D
{
    static void Main()
    {
        var outer = new Outer();
        outer.inner.i = 1;  // CS1648
    }
}
```
