# <a name="compiler-error-cs1032"></a><span data-ttu-id="7a082-102">編譯器錯誤 CS1032</span><span class="sxs-lookup"><span data-stu-id="7a082-102">Compiler Error CS1032</span></span>
<span data-ttu-id="7a082-103">於檔案第一個語彙基元後無法定義或取消定義前置處理器符號</span><span class="sxs-lookup"><span data-stu-id="7a082-103">Cannot define/undefine preprocessor symbols after first token in file</span></span>  
  
 <span data-ttu-id="7a082-104">`#define` 和 `#undef` [前置處理器指示詞](../language-reference/preprocessor-directives/index.md) 必須用於程式開頭的任何其他關鍵字之前 (例如命名空間宣告中所使用的關鍵字)。</span><span class="sxs-lookup"><span data-stu-id="7a082-104">The `#define` and `#undef` [preprocessor directives](../language-reference/preprocessor-directives/index.md) must be used at the beginning of a program, before any other keywords, such as those used in the namespace declaration.</span></span>  
  
 <span data-ttu-id="7a082-105">下列範例會產生 CS1032：</span><span class="sxs-lookup"><span data-stu-id="7a082-105">The following sample generates CS1032:</span></span>  
  
```csharp  
// CS1032.cs  
namespace x  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace  
      public static void Main()  
      {  
      }  
   }  
}  
```
