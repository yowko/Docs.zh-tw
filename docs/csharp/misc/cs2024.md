# 編譯器錯誤 CS2024
無效的檔案區段記憶體對齊數值 '\#'  
  
 已將無效值傳遞給 [\/filealign](../../csharp/language-reference/compiler-options/filealign-compiler-option.md) 編譯器選項。  
  
 下列範例會產生 CS2024：  
  
```  
// CS2024.cs // compile with: /filealign:ex // CS2024 expected class MyClass { public static void Main() { } }  
```
