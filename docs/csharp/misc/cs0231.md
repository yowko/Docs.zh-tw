 <span data-ttu-id="2157f-106">下列範例會產生 CS0231：</span><span class="sxs-lookup"><span data-stu-id="2157f-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs

{

   // To resolve the error, use the following line instead:

   {  
   }  
}  
```
