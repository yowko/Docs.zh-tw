# <a name="compiler-error-cs0744"></a><span data-ttu-id="66f45-102">編譯器錯誤 CS0744</span><span class="sxs-lookup"><span data-stu-id="66f45-102">Compiler Error CS0744</span></span>
<span data-ttu-id="66f45-103">必須是內容關鍵字 'equals'</span><span class="sxs-lookup"><span data-stu-id="66f45-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="66f45-104">`join` 子句的模式是 `join`...`in`...`on`...`equals`，如下列範例所示：</span><span class="sxs-lookup"><span data-stu-id="66f45-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="66f45-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="66f45-105">To correct this error</span></span>  
  
1. <span data-ttu-id="66f45-106">請將 `equals` 關鍵字加入 `join` 子句。</span><span class="sxs-lookup"><span data-stu-id="66f45-106">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="66f45-107">範例</span><span class="sxs-lookup"><span data-stu-id="66f45-107">Example</span></span>  
 <span data-ttu-id="66f45-108">下列程式碼會產生 CS0744：</span><span class="sxs-lookup"><span data-stu-id="66f45-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
