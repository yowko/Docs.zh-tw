# <a name="compiler-error-cs1681"></a>編譯器錯誤 CS1681
您不能重新定義全域外部別名  
  
 全域別名已定義為包含所有非別名參考，因此無法重新定義。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1681：  
  
```csharp  
// CS1681.cs  
// compile with: /reference:global=System.dll  
// CS1681 expected  
  
// try this instead: /reference:System.dll  
class A  
{  
   static void Main() {}  
}  
```
