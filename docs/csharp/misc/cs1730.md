# 編譯器錯誤 CS1730
組件和模組屬性必須位於檔案中所有定義的其他項目之前 \(using 子句與外部別名宣告除外\)。  
  
 套用在組件層級的屬性不能出現在任何類型宣告之後。  
  
### 更正這個錯誤  
  
1.  將屬性移至檔案頂端，但位於 `using` 指示詞和 `extern` 別名宣告之下。  
  
## 範例  
 下列程式碼會產生 CS1730：  
  
```  
// cs1730.cs class Test { } [assembly: System.Attribute] // CS1730  
```
