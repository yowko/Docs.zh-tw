# <a name="compiler-warning-level-1-cs0184"></a>編譯器警告 (層級 1) CS0184

指定的運算式絕不為提供的 ('type') 類型  
  
 運算式絕不為 **true** ，因為您要測試的變數未宣告為 ***type*** 也不是衍生自 ***type***。  
  
 下列範例會產生 CS0184：  
  
```csharp  
// CS0184.cs  
// compile with: /W:1  
class MyClass  
{  
   public static void Main()  
   {  
      int i = 0;  
      if (i is string)   // CS0184  
         i++;  
   }  
}  
```
