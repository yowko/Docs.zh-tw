## <a name="example"></a><span data-ttu-id="625d1-110">範例</span><span class="sxs-lookup"><span data-stu-id="625d1-110">Example</span></span>  
 <span data-ttu-id="625d1-111">下列範例會示範產生 CS1934的一種方法：</span><span class="sxs-lookup"><span data-stu-id="625d1-111">The following example shows one way to produce CS1934:</span></span>  
  
```csharp

using System.Collections;  
static class Test  
{  
    public static void Main()  
    {

        var q = from x in list // CS1934
```  
