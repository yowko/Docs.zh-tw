<span data-ttu-id="cd807-103">C# 語言的設計，就是讓不同文件庫的[基底](../../../csharp/language-reference/keywords/base.md)和衍生類別的版本控制能夠發展兼具回溯相容性。</span><span class="sxs-lookup"><span data-stu-id="cd807-103">The C# language is designed so that versioning between [base](../../../csharp/language-reference/keywords/base.md) and derived classes in different libraries can evolve and maintain backward compatibility.</span></span> <span data-ttu-id="cd807-104">例如，這表示 C# 完全支援在基底[類別](../../../csharp/language-reference/keywords/class.md)中引入與衍生類別成員同名的新成員，不會導致非預期的行為。</span><span class="sxs-lookup"><span data-stu-id="cd807-104">This means, for example, that the introduction of a new member in a base [class](../../../csharp/language-reference/keywords/class.md) with the same name as a member in a derived class is completely supported by C# and does not lead to unexpected behavior.</span></span> <span data-ttu-id="cd807-105">這也表示，類別必須明確指出方法是打算覆寫繼承的方法，還是方法是一種新方法，會隱藏名稱相似的繼承方法。</span><span class="sxs-lookup"><span data-stu-id="cd807-105">It also means that a class must explicitly state whether a method is intended to override an inherited method, or whether a method is a new method that hides a similarly named inherited method.</span></span>  
  
 <span data-ttu-id="cd807-106">在 C# 中，衍生類別可以包含與基底類別方法同名的方法。</span><span class="sxs-lookup"><span data-stu-id="cd807-106">In C#, derived classes can contain methods with the same name as base class methods.</span></span>  
  
-   <span data-ttu-id="cd807-107">基底類別方法必須定義為 [virtual](../../../csharp/language-reference/keywords/virtual.md)。</span><span class="sxs-lookup"><span data-stu-id="cd807-107">The base class method must be defined [virtual](../../../csharp/language-reference/keywords/virtual.md).</span></span>  
  
-   <span data-ttu-id="cd807-108">如果衍生類別中的方法前未加上 [new](../../../csharp/language-reference/keywords/new.md) 或 [override](../../../csharp/language-reference/keywords/override.md) 關鍵字，編譯器就會發出警告，方法會表現為如同有 `new` 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="cd807-108">If the method in the derived class is not preceded by [new](../../../csharp/language-reference/keywords/new.md) or [override](../../../csharp/language-reference/keywords/override.md) keywords, the compiler will issue a warning and the method will behave as if the `new` keyword were present.</span></span>  
  
-   <span data-ttu-id="cd807-109">如果衍生類別中的方法前面加上 `new` 關鍵字，方法會定義為不受基底類別中的方法影響。</span><span class="sxs-lookup"><span data-stu-id="cd807-109">If the method in the derived class is preceded with the `new` keyword, the method is defined as being independent of the method in the base class.</span></span>  
  
-   <span data-ttu-id="cd807-110">如果衍生類別中的方法前面加上 `override` 關鍵字，衍生類別的物件會呼叫該方法，不會呼叫基底類別方法。</span><span class="sxs-lookup"><span data-stu-id="cd807-110">If the method in the derived class is preceded with the `override` keyword, objects of the derived class will call that method instead of the base class method.</span></span>  
  
-   <span data-ttu-id="cd807-111">您可以使用 `base` 關鍵字從衍生類別中呼叫基底類別方法。</span><span class="sxs-lookup"><span data-stu-id="cd807-111">The base class method can be called from within the derived class using the `base` keyword.</span></span>  
  
-   <span data-ttu-id="cd807-112">`override`、`virtual` 和 `new` 關鍵字也可以套用至屬性、索引子和事件。</span><span class="sxs-lookup"><span data-stu-id="cd807-112">The `override`, `virtual`, and `new` keywords can also be applied to properties, indexers, and events.</span></span>  
  
 <span data-ttu-id="cd807-113">C# 方法預設不是虛擬的。</span><span class="sxs-lookup"><span data-stu-id="cd807-113">By default, C# methods are not virtual.</span></span> <span data-ttu-id="cd807-114">如果方法宣告為虛擬，則繼承該方法的任何類別都可以實作自己的版本。</span><span class="sxs-lookup"><span data-stu-id="cd807-114">If a method is declared as virtual, any class inheriting the method can implement its own version.</span></span> <span data-ttu-id="cd807-115">若要使方法成為虛擬的，基底類別的方法宣告中會使用 `virtual` 修飾詞。</span><span class="sxs-lookup"><span data-stu-id="cd807-115">To make a method virtual, the `virtual` modifier is used in the method declaration of the base class.</span></span> <span data-ttu-id="cd807-116">然後，衍生類別可以使用 `override` 關鍵字覆寫基底虛擬方法，或使用 `new` 關鍵字隱藏基底類別中的虛擬方法。</span><span class="sxs-lookup"><span data-stu-id="cd807-116">The derived class can then override the base virtual method by using the `override` keyword or hide the virtual method in the base class by using the `new` keyword.</span></span> <span data-ttu-id="cd807-117">如果不指定 `override` 關鍵字，也不指定 `new` 關鍵字，則編譯器會發出警告，且衍生類別中的方法會隱藏基底類別中的方法。</span><span class="sxs-lookup"><span data-stu-id="cd807-117">If neither the `override` keyword nor the `new` keyword is specified, the compiler will issue a warning and the method in the derived class will hide the method in the base class.</span></span>
