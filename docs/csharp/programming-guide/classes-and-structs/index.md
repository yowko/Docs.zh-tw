 <span data-ttu-id="de354-111">類別是參考型別。</span><span class="sxs-lookup"><span data-stu-id="de354-111">A class is a reference type.</span></span> <span data-ttu-id="de354-112">當建立類別的物件時，物件指派至的變數僅會保留該記憶體的參考。</span><span class="sxs-lookup"><span data-stu-id="de354-112">When an object of the class is created, the variable to which the object is assigned holds only a reference to that memory.</span></span> <span data-ttu-id="de354-113">當物件參考指派至新的變數時，新的變數會參考到原始物件。</span><span class="sxs-lookup"><span data-stu-id="de354-113">When the object reference is assigned to a new variable, the new variable refers to the original object.</span></span> <span data-ttu-id="de354-114">透過某個變數所做的變更會反映在其他變數中，因為它們都參考相同的資料。</span><span class="sxs-lookup"><span data-stu-id="de354-114">Changes made through one variable are reflected in the other variable because they both refer to the same data.</span></span>  
  
 <span data-ttu-id="de354-115">結構是實值型別。</span><span class="sxs-lookup"><span data-stu-id="de354-115">A struct is a value type.</span></span> <span data-ttu-id="de354-116">當建立結構時，結構指派至的變數會保留結構的實際資料。</span><span class="sxs-lookup"><span data-stu-id="de354-116">When a struct is created, the variable to which the struct is assigned holds the struct's actual data.</span></span> <span data-ttu-id="de354-117">將結構指派至新的變數時，將會複製結構。</span><span class="sxs-lookup"><span data-stu-id="de354-117">When the struct is assigned to a new variable, it is copied.</span></span> <span data-ttu-id="de354-118">因此，新的變數和原始變數會各自包含一份相同的資料。</span><span class="sxs-lookup"><span data-stu-id="de354-118">The new variable and the original variable therefore contain two separate copies of the same data.</span></span> <span data-ttu-id="de354-119">針對其中一個複本所做的變更，並不會影響到另一個複本。</span><span class="sxs-lookup"><span data-stu-id="de354-119">Changes made to one copy do not affect the other copy.</span></span>  
  
 <span data-ttu-id="de354-120">一般情況下，類別會用來建立更複雜行為的模型，或是要在建立類別物件後修改之資料的模型。</span><span class="sxs-lookup"><span data-stu-id="de354-120">In general, classes are used to model more complex behavior, or data that is intended to be modified after a class object is created.</span></span> <span data-ttu-id="de354-121">結構最適合小型資料結構，該結構主要包含不會在結構建立後修改的資料。</span><span class="sxs-lookup"><span data-stu-id="de354-121">Structs are best suited for small data structures that contain primarily data that is not intended to be modified after the struct is created.</span></span>
