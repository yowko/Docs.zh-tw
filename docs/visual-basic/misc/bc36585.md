# &#39;&lt;項目名稱&gt;&#39; 不是 &#39;&lt;類型名稱&gt;&#39; 中所定義擴充方法 &#39;&lt;方法名稱&gt;&#39; 的參數
程序呼叫可提供擴充方法未定義的其他引數。  
  
 **錯誤 ID︰**BC36585  
  
### 更正這個錯誤  
  
-   請從引數清單中移除多餘的引數。
