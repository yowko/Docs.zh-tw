# &#39;#Region&#39; 陳述式之後必須搭配相對應的 &#39;#End Region&#39;
使用 `#Region` 指示詞來指定程式碼區段，以在使用 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] 程式碼編輯器的大綱功能時展開或摺疊。`#Region` 陳述式的開頭和結尾必須在相同的程式碼區塊中。  
  
 **錯誤 ID：**BC30681  
  
### 更正這個錯誤  
  
1.  請在 `#Region` 陳述式後面之程式碼的適當位置插入 `#End Region`。
