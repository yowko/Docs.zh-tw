# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="55c50-102">'Implements' 子句中不允許類型參數</span><span class="sxs-lookup"><span data-stu-id="55c50-102">Type parameter not allowed in 'Implements' clause</span></span>
<span data-ttu-id="55c50-103">泛型類型中的 `Implements` 子句指定類型參數作為要實作的成員。</span><span class="sxs-lookup"><span data-stu-id="55c50-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="55c50-104">`Implements` 子句必須指定介面和成員。</span><span class="sxs-lookup"><span data-stu-id="55c50-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="55c50-105">它可以將類型參數傳遞給介面，但無法將它傳遞給成員，或使用它作為成員的名稱。</span><span class="sxs-lookup"><span data-stu-id="55c50-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="55c50-106">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="55c50-106">The following statements can generate this error.</span></span>  
  
```  
Class c1(Of t)  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="55c50-107">**錯誤 ID:** BC32056</span><span class="sxs-lookup"><span data-stu-id="55c50-107">**Error ID:** BC32056</span></span>
