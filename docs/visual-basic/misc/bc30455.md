# &#39;&lt;方法名稱&gt;&#39; 的參數 &#39;&lt;參數名稱&gt;&#39; 未指定引數
尚未提供必要參數的引數。  
  
 **錯誤 ID︰**BC30455  
  
### 更正這個錯誤  
  
-   請為指定的參數提供引數。
