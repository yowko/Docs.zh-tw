# 物件不支援此屬性或方法 (Visual Basic)
您指定的方法或屬性對於此自動化物件不存在。  
  
### 更正這個錯誤  
  
-   請閱讀物件的文件取得物件的詳細資訊，並檢查屬性和方法的拼字。
