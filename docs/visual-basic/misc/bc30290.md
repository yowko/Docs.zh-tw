# 區域變數不可以和包含它的函式具有相同名稱
`Dim` 陳述式所指定變數的名稱與包含它的 `Function` 程序相同。  
  
 **錯誤 ID︰**BC30290  
  
### 更正這個錯誤  
  
-   請移除變數宣告，或變更變數的名稱。
