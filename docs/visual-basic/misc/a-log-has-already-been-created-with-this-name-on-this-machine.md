# 已在這部電腦上建立具有這個名稱的記錄檔。
嘗試使用已在使用的名稱來建立記錄檔。  
  
### 更正這個錯誤  
  
-   移除衝突的記錄檔。  
  
-   使用不同的記錄檔名稱。
