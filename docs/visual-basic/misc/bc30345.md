# &#39;&lt;方法 1&gt;&#39; 和 &#39;&lt;方法 2&gt;&#39; 無法互相多載，因為它們的差異只在於宣告為 &#39;ByRef&#39; 或 &#39;ByVal&#39; 的參數不同而已
您已嘗試將某個方法多載另一個方法，而且這兩個方法的差異只在於宣告為 `ByRef` 或 `ByVal` 的參數不同。  
  
 **錯誤 ID：**BC30345  
  
### 更正這個錯誤  
  
-   確定方法的區分方式不只是透過 `ByRef` 或 `ByVal` 參數的名稱。
