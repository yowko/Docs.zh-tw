# <a name="expected-closing-39gt39-for-embedded-expression"></a>預期結尾 &#39; %&gt;&#39; 對於內嵌的運算式
XML 常值中的內嵌運算式遺漏必要的結尾 `%>` 字元。  
  
 **錯誤 ID︰** BC31159  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將必要的結尾 `%>` 字元加入內嵌運算式的結尾。
