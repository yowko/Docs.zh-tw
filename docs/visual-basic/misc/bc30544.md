# 方法不可以同時包含 &#39;Try&#39; 陳述式以及 &#39;On Error&#39; 或 &#39;Resume&#39; 陳述式
您已搭配 `On Error` 或 `Resume` 使用 `Try` 陳述式。  
  
 **錯誤 ID︰**BC30544  
  
### 更正這個錯誤  
  
1.  移除 `On Error` 或 `Resume`。
