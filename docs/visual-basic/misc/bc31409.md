# <a name="ltmethodgt-ltmethodname1gt-must-be-declared-overloads-because-another-ltmethodname2gt-is-declared-overloads"></a>&lt;方法&gt;'&lt;methodname1&gt;' 必須宣告為 'Overloads' 因為另一個 '&lt;methodname2&gt;' 已宣告為 'Overloads'
使用 `Overloads` 修飾詞宣告同名的另一個屬性或方法。  
  
 **錯誤 ID:** BC31409  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  將 `Overloads` 修飾詞加入屬性或方法。  
  
2.  從同名的所有屬性或方法中移除 `Overloads` 修飾詞。
