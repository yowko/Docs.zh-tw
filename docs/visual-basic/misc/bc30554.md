# &#39;&lt;命名空間名稱&gt;&#39; 模稜兩可
您提供的名稱模稜兩可，因此與另一個名稱發生衝突。[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 編譯器並沒有任何衝突解決規則，您必須自行區分名稱。  
  
 **錯誤 ID︰**BC30554  
  
### 更正這個錯誤  
  
-   完整限定名稱。
