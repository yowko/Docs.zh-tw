# 此內容中不允許 &#39;Group&#39;; 必須是識別項
`Group` 關鍵字包含在 `Aggregate` 子句的 `Into` 區段。`Group` 關鍵字只在 `Group By` 或 `Group Join` 子句中有效。  
  
 **錯誤 ID︰**BC36618  
  
### 更正這個錯誤  
  
-   請從 `Aggregate` 子句移除 `Group` 關鍵字。 您可以將 `Group By` 子句加入查詢，以將結果分組。
