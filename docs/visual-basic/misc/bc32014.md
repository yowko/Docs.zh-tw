# 在初始化陣列的陣列時，只可以指定最上層陣列的界限
不規則陣列 \(陣列的陣列\) 的陣列初始化會設定其中一個較低層級的初始長度。 您只能在陣列宣告陳述式中指定最上層陣列的長度。  
  
 **錯誤 ID︰**BC32014  
  
### 更正這個錯誤  
  
1.  請移除最上層陣列以外的所有長度指定。  
  
2.  請在後續的指派陳述式中使用 `New` 關鍵字設定較低層級陣列的初始長度。
