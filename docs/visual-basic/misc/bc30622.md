# &#39;End Module&#39; 之前必須搭配相對應的 &#39;Module&#39;
模組區塊會以 `Module` 陳述式開頭並以 `End Module` 建構結尾。  
  
 **錯誤 ID：**BC30622  
  
### 更正這個錯誤  
  
-   確定模組區塊以 `Module` 陳述式開頭。
