# 方法引數必須放在括弧之中
管理程序呼叫的規則在較新版本的 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 中比較簡單。 所有引數必須放在括弧之中。  
  
 **錯誤 ID︰**BC30800  
  
### 更正這個錯誤  
  
-   請將引數清單用括弧括住。例如：  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
