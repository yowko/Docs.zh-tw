# 在此內容中的運算式評估期間，副作用無效
在此內容中，不允許變更物件狀態的陳述式。  
  
 **錯誤 ID︰**BC30700  
  
### 更正這個錯誤  
  
-   請變更您的程式碼，以評估陳述式而不變更其他項目的狀態。
