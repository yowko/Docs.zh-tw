# 在多個基底介面都有 &#39;&lt;成員名稱&gt;&#39;
在多個基底介面都有 '\<成員名稱\>'。 請使用在 'Implements' 子句中宣告 '\<成員名稱\>' 的介面名稱來取代衍生介面的名稱。  
  
 這個介面會從多個介面中繼承同名成員，進而產生模稜兩可。  
  
 **錯誤 ID︰**BC31040  
  
### 更正這個錯誤  
  
-   在 `Implements` 子句中使用定義介面名稱，而非衍生介面的名稱。
