# <a name="member-ltmembernamegt-cannot-be-initialized-in-an-object-initializer-expression-because-it-is-not-a-field-or-property"></a>成員 '&lt;membername&gt;' 無法在物件初始設定式運算式中初始化，因為它不是欄位或屬性
物件初始設定式清單不能包括共用成員、常數或方法呼叫。 初始設定式清單中的成員必須是欄位或屬性，而且屬性成員不需要引數。  
  
 **錯誤 ID:** BC30990  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從物件初始設定式清單中移除所有共用成員、常數、方法呼叫或具有參數的屬性。
