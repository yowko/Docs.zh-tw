# &#39;&lt;成員名稱1&gt;&#39; 無法覆寫 &#39;&lt;成員名稱2&gt;&#39;，因為其選擇性參數類型不同
多載成員的一或多個必要參數必須具有不同的資料類型。  
  
 **錯誤 ID︰**BC30697  
  
### 更正這個錯誤  
  
-   請至少將一個必要的參數定義為不同的資料類型。
