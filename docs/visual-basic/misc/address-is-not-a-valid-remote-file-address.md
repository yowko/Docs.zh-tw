# &lt;位址&gt; 不是有效的遠端檔案位址
\<位址\> 不是有效的遠端檔案位址。 有效位址應該包含通訊協定、路徑和檔案名稱。  
  
 針對上傳或下載作業指定了無效的位址。 位址必須包括通訊協定 \(例如 `ftp` 或 `http`\)、有效的路徑和檔案名稱。 這個錯誤可能是不正確剖析檔案路徑的結果。  
  
### 更正這個錯誤  
  
-   檢查您所指定的位址以判斷失敗的位置，並提供必要的資訊。
