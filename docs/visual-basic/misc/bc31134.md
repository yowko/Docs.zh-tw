# &#39;AddHandler&#39; 和 &#39;RemoveHandler&#39; 方法參數不可以宣告為 &#39;ByRef&#39;
不能使用 `ByRef` 修飾詞來宣告 `AddHandler` 和 `RemoveHandler` 方法的參數。  
  
 **錯誤 ID︰**BC31134  
  
### 更正這個錯誤  
  
-   請從參數中移除 `ByRef` 關鍵字。
