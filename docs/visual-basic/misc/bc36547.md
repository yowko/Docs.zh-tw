<span data-ttu-id="5be1e-103">屬性名稱只能在匿名類型的宣告中建立一次。</span><span class="sxs-lookup"><span data-stu-id="5be1e-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="5be1e-104">例如，下列宣告無效：</span><span class="sxs-lookup"><span data-stu-id="5be1e-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.

'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```
