# 運算子不是可多載的
只有特定運算子適用於多載。 下表列出您可以定義的運算子清單。  
  
|類型|運算子|  
|--------|---------|  
|一元|`+`, `-`, `IsFalse`, `IsTrue`, `Not`|  
|二元|`+`, `-`, `*`, `/`, `\`, `&`, `^`, `>>`, `<<`, `=`, `<>`, `>`, `>=`, `<`, `<=`, `And`, `Like`, `Mod`, `Or`, `Xor`|  
|轉換 \(一元\)|`CType`|  
  
 請注意，二元清單中的 `=` 運算子是比較運算子，而不是指派運算子。  
  
 **錯誤 ID︰**BC33002  
  
### 更正這個錯誤  
  
1.  從一組可多載的運算子中選取運算子。  
  
2.  如果您需要多載無法直接多載之運算子的功能，請建立接受適當參數並傳回適當值的 `Function` 程序。
