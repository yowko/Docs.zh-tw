# &#39;Case Else&#39; 只能在 &#39;Select Case&#39; 陳述式內出現
`Case Else` 陳述式出現在 `Select` 區塊外。`Case Else` 陳述式只能用在 `Select` 或 `Select Case` 陳述式和其相對應的 `End Select` 陳述式之間。  
  
 **錯誤 ID：**BC30071  
  
### 更正這個錯誤  
  
-   移除 `Case Else` 陳述式或將它移至 `Select` 區塊內。
