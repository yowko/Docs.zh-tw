# &#39;AddHandler&#39; 和 &#39;RemoveHandler&#39; 方法參數必須與包含事件擁有相同的委派類型
`Custom Event` 宣告必須有 `AddHandler` 或 `RemoveHandler` 宣告，每個宣告會接受自訂事件的 `As` 子句所指定之委派類型的單一參數。  
  
 **錯誤 ID︰**BC31136  
  
### 更正這個錯誤  
  
-   請變更參數類型，讓它與自訂事件之 `As` 子句所指定的委派類型相同。
