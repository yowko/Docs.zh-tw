# 在專案 &#39;&lt;專案名稱1&gt;&#39; (不是由專案 &#39;&lt;專案名稱2&gt;&#39; 參考) 中宣告的 &#39;&lt;名稱&gt;&#39;
程式設計項目定義於目前專案未參考的專案中。  
  
 **錯誤 ID︰**BC32004  
  
### 更正這個錯誤  
  
-   在目前專案中，加入包含所要項目之專案的參考。
