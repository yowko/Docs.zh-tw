# XML 格式不正確。 不能包含檔案 &#39;&lt;註解檔案&gt;&#39;
XML 註解中其中一個 `<include>` 標記參考了不存在的檔案或檔案中的 XML 格式不正確。  
  
 **錯誤 ID：**BC42321  
  
### 更正這個錯誤  
  
1.  請確認檔案存在。  
  
2.  請確認檔案中的 XML 有效。
