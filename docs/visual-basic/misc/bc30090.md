# &#39;End While&#39; 之前必須搭配相對應的 &#39;While&#39;
`End While` 陳述式沒有相對應的 `While` 陳述式。`End While` 之前必須搭配相對應的 `While` 陳述式。  
  
 **錯誤 ID：**BC30090  
  
### 更正這個錯誤  
  
1.  如果這個 `While` 區塊是一組巢狀 `While` 區塊的一部分，請確定已正確地終止每個區塊。  
  
2.  請確認已正確地終止 `While` 區塊內的其他控制結構。  
  
3.  請確定已正確地格式化這個 `While` 區塊。
