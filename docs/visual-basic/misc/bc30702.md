# 維度 &#39;&lt;維度編號&gt;&#39; 的索引 &#39;&lt;索引編號&gt;&#39; 超出範圍
其中一個索引超出定義的範圍。  
  
 **錯誤 ID︰**BC30702  
  
### 更正這個錯誤  
  
-   請變更程序的邏輯，使其不超出陣列界限。  
  
-   重新定義陣列，以允許較大範圍的值。
