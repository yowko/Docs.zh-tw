# &#39;&lt;成員名稱&gt;&#39; 與類型參數擁有相同的名稱
泛型類別或結構的成員與該類別或結構的類型參數同名。  
  
 每個變數、常數、列舉、程序、屬性、事件和委派的名稱都必須與每個類型參數不同。  
  
 **錯誤 ID︰**BC32054  
  
### 更正這個錯誤  
  
-   請變更類別或結構成員的名稱，使其與每個類型參數名稱不同。
