# &#39;ReDim&#39; 無法變更陣列的維度數目
您嘗試使用 `ReDim` 變更陣列的陣序 \(維度數目\)。`ReDim` 陳述式可用來變更已正式宣告之陣列的一或多個維度大小，但無法變更陣列的陣序。  
  
 **錯誤 ID︰**BC30415  
  
### 更正這個錯誤  
  
-   確定您指定陣序，而不是陣列的大小，並盡可能使用 `Dim` 來宣告具有所需陣序的新陣列。
