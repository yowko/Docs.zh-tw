# 指定的登錄機碼無效，因為它包含兩個以上的連續反斜線
指定具有路徑的登錄機碼包含兩個以上的連續反斜線。 原因可能是合併數個字串來組成路徑，但意外地包括太多反斜線。  
  
### 更正這個錯誤  
  
-   請檢查所指定的登錄機碼，以判定額外反斜線的插入位置和插入原因。
