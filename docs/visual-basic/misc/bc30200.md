# &#39;New&#39; 在此內容中無效
需要類型名稱的位置出現 `New` 關鍵字。 這個內容呼叫無法與 `New` 合併的實值類型。  
  
 **錯誤 ID︰**BC30200  
  
### 更正這個錯誤  
  
-   請移除 `New` 關鍵字。
