# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>不再支援 'Currency' 類型。請使用 'Decimal' 類型代替
`Currency` 資料類型取代了 `Decimal` 資料類型。  
  
 **錯誤 ID:** BC30803  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `Decimal` 資料類型取代 `Currency`。
