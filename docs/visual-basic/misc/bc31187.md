# 命名空間宣告必須要使用 &#39;xmlns&#39; 開頭
已指定 XML 命名空間，但不含必要的 `xmlns` 識別項。`xmlns` 識別項只能包含小寫字元。  
  
 **錯誤 ID：**BC31187  
  
### 更正這個錯誤  
  
-   當您宣告 XML 命名空間時，請使用 `xmlns` 識別碼。 以下是一個範例：  
  
    ```vb#  
    Imports <xmlns:ns="http://SampleNamespace">  
    ```
