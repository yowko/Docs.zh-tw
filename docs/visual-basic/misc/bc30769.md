# .NET Compact Framework 的目標版本不支援 &#39;End&#39; 陳述式
您使用的 .NET Compact Framework 版本不支援 `End` 陳述式。  
  
 **錯誤 ID︰**BC30769  
  
### 更正這個錯誤  
  
-   請移除陳述式。
