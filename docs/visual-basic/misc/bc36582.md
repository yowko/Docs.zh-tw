# 在 &#39;&lt;類型名稱&gt;&#39; 中定義之擴充方法 &#39;&lt;方法名稱&gt;&#39; 的引數太多
擴充方法呼叫中所提供的引數數目超過為方法所定義的引數數目。  
  
 **錯誤 ID︰**BC36582  
  
### 更正這個錯誤  
  
-   請參考擴充方法的定義，並從呼叫移除多餘的引數。
