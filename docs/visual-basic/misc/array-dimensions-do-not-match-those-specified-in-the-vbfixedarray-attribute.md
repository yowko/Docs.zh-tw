# 陣列維度與 &#39;VBFixedArray&#39; 屬性所指定的維度不相符。
您的陣列維度不符合 `VBFixedArray` 屬性所指定的維度。  
  
### 更正這個錯誤  
  
-   請確定陣列 `Dim` 陳述式中的陣列維度符合 `VBFixedArray` 屬性所指定的維度。
