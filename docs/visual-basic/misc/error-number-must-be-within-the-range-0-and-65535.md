# <a name="error-number-must-be-within-the-range-0-and-65535"></a>錯誤號碼的範圍必須落在 0 與 65535 內
錯誤號碼落在 0 與 65535 之間的允許值外部。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請確定號碼落在有效範圍內。
