# .NET Compact Framework 的目標版本不支援使用 Ansi、Auto 或 Unicode 修飾詞
您使用的 .NET Compact Framework 版本不支援 `Ansi`、`Auto` 和 `Unicode` 修飾詞。  
  
 **錯誤 ID︰**BC30763  
  
### 更正這個錯誤  
  
-   請移除修飾詞。
