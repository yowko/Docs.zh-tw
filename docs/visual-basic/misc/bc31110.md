# 明確指定所有項目值時，當做屬性引數的陣列是必要的
當做屬性引數使用的陣列中具有未指定值的項目。  
  
 **錯誤 ID︰**BC31110  
  
### 更正這個錯誤  
  
-   指定所有項目值。
