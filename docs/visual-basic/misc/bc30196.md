# <a name="comma-expected"></a>必須有逗號
引數清單中的兩個引數之間遺漏逗號 (,)。  
  
 **錯誤 ID:** BC30196  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定所有引數皆以逗號分隔。
