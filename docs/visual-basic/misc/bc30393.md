# &#39;Exit Try&#39; 只能在 &#39;Try&#39; 陳述式內出現
`Exit Try` 只可以出現在 `Try` 區塊陳述式內。 您有多餘的 `Exit Try` 陳述式，或您的 `Exit Try` 陳述式出現在其對應 `Try` 區塊的範圍之外。  
  
 **錯誤 ID︰**BC30393  
  
### 更正這個錯誤  
  
1.  找到並移除不必要的`Exit Try` 陳述式。  
  
2.  請將 `Exit Try`陳述式移至您程式碼中的適當位置。
