# 無法開啟模組檔 &#39;&lt;檔案名稱&gt;&#39;：&lt;錯誤&gt;
無法開啟包含此專案之模組的檔案。  
  
 **錯誤 ID︰**BC31007  
  
### 更正這個錯誤  
  
1.  請確認檔案位於指定的位置，且未被其他應用程式鎖定。
