# <a name="variable-cannot-be-initialized-with-non-array-type-ltelementnamegt"></a>無法以非陣列類型初始化變數 '&lt;elementname&gt;'
必須使用陣列值來初始化宣告為陣列的變數。  
  
```  
' Not valid.  
' The following line causes this error when executed with Option Strict off.  
' Dim arrayVar1() = 10  
```  
  
 **錯誤 ID:** BC36536  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請使用陣列值來初始化陣列變數：  
  
    ```
    ' With Option Strict off.  
    Dim arrayVar2() = {1, 2, 3}  
    ' With Option Strict on.  
    Dim arrayVar3() As Integer = {1, 2, 3}  
    ```
