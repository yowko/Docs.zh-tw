# <a name="variable-cannot-be-initialized-with-non-array-type-elementname"></a>無法使用非陣列類型 ' @no__t 0elementname > ' 初始化變數
必須使用陣列值來初始化宣告為陣列的變數。  
  
```vb  
' Not valid.  
' The following line causes this error when executed with Option Strict off.  
' Dim arrayVar1() = 10  
```  
  
 **錯誤識別碼：** BC36536  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請使用陣列值來初始化陣列變數：  
  
    ```vb
    ' With Option Strict off.  
    Dim arrayVar2() = {1, 2, 3}  
    ' With Option Strict on.  
    Dim arrayVar3() As Integer = {1, 2, 3}  
    ```
