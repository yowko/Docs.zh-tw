# &#39;&lt;方法1&gt;&#39; 和 &#39;&lt;方法2&gt;&#39; 無法互相多載，因為它們的差異只在於傳回類型不同而已。
您已嘗試將某個方法多載另一個方法，後者只有傳回類型和前者不同。 在多載中，您必須藉引數清單區分任何兩個版本的方法；除了引數清單，沒有任何其他項目可以區別方法。  
  
 **錯誤識別碼：**BC30301  
  
### 更正這個錯誤  
  
-   請確定方法是以其引數清單區分。
