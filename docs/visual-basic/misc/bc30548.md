# 無法將屬性 &#39;&lt;屬性名稱&gt;&#39; 套用至組件
您嘗試將屬性套用至其 `AttributeUsageAttribute` 未指定 `AttributeTargets.Assembly` 的組件。 當宣告屬性時，不會將其定義為適用於組件。  
  
 **錯誤 ID︰**BC30548  
  
### 更正這個錯誤  
  
1.  檢查屬性宣告，並指定 `AttributeTargets.Assembly` 或 `AttributeTargets.All`。
