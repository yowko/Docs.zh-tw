# XML 標記必須以 &#39;&lt;&#39; 開頭
XML 常值的 XML 標記缺少必要的開頭 '\<' 字元。  
  
 **錯誤 ID︰**BC31165  
  
### 更正這個錯誤  
  
-   請在 XML 常值的 XML 標記開頭加入 '\<' 字元。
