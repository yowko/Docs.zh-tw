# 只可以指定 &#39;NotOverridable&#39;、&#39;MustOverride&#39; 或 &#39;Overridable&#39; 其中之一
項目宣告中發生覆寫規格衝突。 覆寫關鍵字互斥。  
  
 **錯誤 ID︰**BC30177  
  
### 更正這個錯誤  
  
-   請選擇一個覆寫規範，並移除宣告中的其他關鍵字。
