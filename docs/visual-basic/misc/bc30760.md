# 參考的物件具有 &#39;Nothing&#39; 的值
正在使用的物件具有值 `Nothing`，但必須是可使用的值。`Nothing` 是一個值，表示物件沒有值，這可能是因為尚未指定任何值給它，或它被指派值 `Nothing`。  
  
 **錯誤 ID︰**BC30760  
  
### 更正這個錯誤  
  
1.  請檢查物件，並確定它已在發生錯誤的程序範圍內宣告。  
  
2.  確認要設定之物件的值。  
  
    > [!NOTE]
    >  值 `Nothing` 不為零或空字串。 您可以使用 `IsNothing` 來檢查物件是否包含值 `Nothing`。
