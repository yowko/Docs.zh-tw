# 無法包含檔案 &#39;&lt;註解檔案&gt;&#39; 的 XML 片段 &#39;&lt;項目路徑&gt;&#39;
XML 註解中的 `<include>` 標記包含不存在之項目的 XPath 運算式，或檔案包含格式不正確的 XML。  
  
 **錯誤 ID︰**BC42320  
  
### 更正這個錯誤  
  
1.  請確認檔案中的 XML 有效。  
  
2.  請確認 XML 片段存在於檔案中。
