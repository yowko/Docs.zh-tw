# <a name="leading--or--cannot-appear-in-a-constant-expression"></a>前置的 '.' 或 '!' 不能出現在常數運算式中
成員存取（.）和字典成員存取（！）需要一個運算式，指定包含最多時間成員的元素，包括常數運算式。 下列宣告無效。  
  
```vb  
' Not valid.  
Const c As String = .name  
```  
  
 **錯誤識別碼：** BC30995  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請指定包含您想要存取之成員的執行個體。
