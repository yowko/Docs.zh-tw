# 不再支援 &#39;Let&#39; 和 &#39;Set&#39; 指派陳述式
`Let` 和 `Set` 不可以用於指派中。  
  
 **錯誤 ID︰**BC30807  
  
### 更正這個錯誤  
  
-   指派陳述式中省略 `Let` 和 `Set`；例如：  
  
     `X = 2`
