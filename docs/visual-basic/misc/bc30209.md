# Option Strict On 要求所有的變數宣告皆須有 &#39;As&#39; 子句
宣告中包含宣告的變數，但不含 `As` 子句。 當 `Option Strict` 是 `On` 時，必須使用 `As` 子句宣告每個變數、屬性、程序引數與函式傳回以指定其資料類型，例如 `Dim MyNum As Short`。  
  
 **錯誤 ID：**BC30209  
  
### 更正這個錯誤  
  
1.  請查看 `As` 關鍵字是否拼字錯誤。  
  
2.  針對宣告的變數提供 `As` 子句，或轉為 `Option Strict Off`。
