# Function 或 Property 中的 &#39;Exit Sub&#39; 無效
`Exit Sub` 出現在 `Function` 程序或 `Property` 程序。`Exit` 陳述式必須符合它發生所在的區塊。  
  
 **錯誤 ID︰**BC30065  
  
### 更正這個錯誤  
  
-   請視需要將 `Exit Sub` 取代為 `Exit Function` 或 `Exit Property` 陳述式。
