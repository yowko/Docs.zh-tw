# 控制代碼中不允許 &#39;Global&#39;; 必須是區域名稱
`Handles` 子句必須參考本機的事件。`Global` 關鍵字可提供對通用程式設計項目的存取。  
  
 **錯誤 ID︰**BC36002  
  
### 更正這個錯誤  
  
-   請變更 `Handles` 子句以參考事件的本機執行個體，而不要參考全域執行個體。
