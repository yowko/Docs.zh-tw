# 屬性 &lt;屬性名稱&gt; 不可以設定為空字串或 Nothing
此屬性需要空字串 \(""\) 或 `Nothing` 以外的值。 這可能是不正確計算值的結果。  
  
### 更正這個錯誤  
  
-   請提供有效的值。
