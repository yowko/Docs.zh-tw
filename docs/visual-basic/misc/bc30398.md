# &#39;&lt;方法1&gt;&#39; 無法覆寫 &#39;&lt;方法2&gt;&#39;，因為它們的差異只在於參數標記為 &#39;ByRef&#39; 或 &#39;ByVal&#39; 而已
您嘗試使用與已標記 `ByRef` \(非 `ByVal`\) 之參數不同的另一種方法來覆寫方法。 覆寫的成員必須具有與基底類別之繼承成員相同的引數。  
  
 **錯誤 ID︰**BC30398  
  
### 更正這個錯誤  
  
-   請確定參數都是 `ByRef` 或都是 `ByVal`。
