# 無法在偵錯視窗中評估 Web 方法
在偵錯視窗內無法測試 XML Web 服務方法。  
  
 **錯誤 ID︰**BC30732  
  
### 更正這個錯誤  
  
-   從程式碼編輯器，測試 XML Web 服務方法。
