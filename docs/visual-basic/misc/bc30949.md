# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="ad8b5-102">無法為非常數維度指定陣列初始化運算式;使用空的初始化運算式 ' {} '</span><span class="sxs-lookup"><span data-stu-id="ad8b5-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>
<span data-ttu-id="ad8b5-103">陣列初始化編譯時期未知的維度。</span><span class="sxs-lookup"><span data-stu-id="ad8b5-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="ad8b5-104">下列程式碼會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="ad8b5-104">The following code generates this error.</span></span>  
  
```vb  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="ad8b5-105">下列程式碼可避免這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="ad8b5-105">The following code avoids the error.</span></span>  
  
```vb  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="ad8b5-106">**錯誤識別碼：** BC30949</span><span class="sxs-lookup"><span data-stu-id="ad8b5-106">**Error ID:** BC30949</span></span>
