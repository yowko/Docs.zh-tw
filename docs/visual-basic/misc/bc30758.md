# 無法使用屬性，因為它沒有 Public 建構函式
所要使用的屬性建構函式為 `Private`，但無法使用。  
  
 **錯誤 ID︰**BC30758  
  
### 更正這個錯誤  
  
1.  如果您所開發的自訂屬性發生這個錯誤，請將其 `Sub``New` 建構函式的存取修飾詞變更為 `Public`。
