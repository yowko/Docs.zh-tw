# 無法將 &#39;&lt;類型名稱&gt;&#39; 當做屬性使用，因為它擁有尚未被覆寫的 &#39;MustOverride&#39; 方法
無法將具有 `MustOverride` 方法的類別當做屬性使用。  
  
 屬性類別的 `MustOverride` 成員只能透過覆寫這類成員的衍生類別使用。  
  
 **錯誤 ID︰**BC31507  
  
### 更正這個錯誤  
  
1.  從屬性類別成員移除 `MustOverride` 修飾詞。  
  
2.  在衍生類別中實作 `MustOverride` 成員，並使用該類別作為屬性。
