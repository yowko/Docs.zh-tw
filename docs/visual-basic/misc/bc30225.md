# &#39;Option &lt;規範&gt;&#39; 陳述式只可以在每個檔案出現一次
`Option Compare`、`Option Explicit` 和 `Option Strict` 陳述式可能在相同的原始程式檔中各使用一次。  
  
 **錯誤 ID︰**BC30225  
  
### 更正這個錯誤  
  
-   請移除重複的 `Option` 陳述式。
