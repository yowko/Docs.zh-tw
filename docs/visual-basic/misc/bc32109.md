# <a name="indirect-constraint-constraint1-obtained-from-the-type-parameter-constraint-typeparameter1-conflicts-with-the-indirect-constraint-constraint2-obtained-from-the-type-parameter-constraint-typeparameter2"></a>從類型參數條件約束 ' \<typeparameter1 > ' 取得的間接條件約束 ' @no__t 0constraint1 > ' 與從類型參數條件約束 ' ' 取得的間接條件約束 ' \<constraint2 > ' 相衝突 \<typeparameter2 > '
宣告的泛型類型因間接條件約束組合而具有衝突的條件約束。  
  
 下列陳述式可能會產生這個錯誤。  
  
```vb  
Public Class testClass(Of t1 As {t2, t3}, t2 As Structure, t3 As Class)  
```  
  
 類型參數 `Structure` 的間接條件約束 `Class` 和 `t1`會導致衝突，因為 `Structure` 條件約束需要對應的類型參數是實值類型，而 `Class` 需要它是參考類型。  
  
 **錯誤識別碼：** BC32109  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請變更類型參數條件約束，以避免衝突的條件約束。
