# <a name="missing-required-white-space"></a>遺漏必要的空白字元
已指定沒有必要分隔空間的 XML 常值。 必要分隔空間範例是 XML 項目常值中後續屬性之間的空間。  
  
 **錯誤 ID:** BC31173  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請加入必要分隔空間。
