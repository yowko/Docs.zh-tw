# &#39;&lt;成員名稱&gt;&#39; 未宣告 或包含它的模組未於偵錯工作階段中載入
找不到指定的成員。  
  
 **錯誤 ID︰**BC30699  
  
### 更正這個錯誤  
  
1.  請確認已定義成員，而且拼寫正確。  
  
2.  存取模組中所宣告的其中一個成員。 在某些情況下，偵錯環境找不到成員，因為尚未載入宣告它們的模組。
