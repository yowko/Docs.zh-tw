# &#39;ReDim&#39; Preserve 不可以是 Nothing
`ReDim` 陳述式嘗試使用 `Preserve` 關鍵字來變更不是最後一個維度的陣列維度，但未提供其運算元的有效值。  
  
### 更正這個錯誤  
  
-   將 `Preserve` 運算元變更為有效值。
