# &#39;&lt;修飾詞&gt;&#39; 在區域變數宣告中無效
程序內的 `Dim` 陳述式包含無效的關鍵字，例如 `Overloads`。  
  
 **錯誤 ID︰**BC30247  
  
### 更正這個錯誤  
  
-   請從 `Dim` 陳述式中移除無效的關鍵字。
