# <a name="modifier-is-not-valid-on-a-local-variable-declaration"></a>'\<修飾詞 >' 在區域變數宣告中無效
程序內的 `Dim` 陳述式包含無效的關鍵字，例如 `Overloads`。  
  
 **錯誤 ID:** BC30247  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請從 `Dim` 陳述式中移除無效的關鍵字。
