 <span data-ttu-id="dbe63-104">在某些情況下，編譯器可決定 `TypeOf`...`Is` 測試只會失敗；例如，如果兩個類別之間沒有任何繼承關聯性。</span><span class="sxs-lookup"><span data-stu-id="dbe63-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>

 `End If`
