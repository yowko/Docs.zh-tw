# <a name="39ltnamegt39-is-already-declared-as-a-parameter-of-this-method"></a>&#39;&lt;名稱&gt;&#39; 已經宣告為此方法的參數
已經針對這個方法宣告指定的參數名稱。  
  
 **錯誤 ID︰** BC30734  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   重新命名參數，使其變成唯一的。
