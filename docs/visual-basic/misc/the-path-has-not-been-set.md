# 尚未設定路徑
嘗試檔案作業 \(例如移動或複製檔案\)，但未提供必要的路徑。 這可能是不正確剖析路徑的結果。  
  
### 更正這個錯誤  
  
-   確認已提供檔案路徑，而且正確無誤。
