# &#39;Inherits&#39; 陳述式必須位在類別所有宣告之前。
`Inherits` 陳述式必須位在類別定義的開頭。  
  
 **錯誤識別碼：**BC30683  
  
### 更正這個錯誤  
  
-   將 `Inherits` 陳述式移至類別的頂端。
