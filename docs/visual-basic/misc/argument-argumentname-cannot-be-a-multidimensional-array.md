# <a name="argument-argumentname-cannot-be-a-multidimensional-array"></a>引數 '\<引數名稱 >' 不可以是多維陣列
array 引數的順位大於 1，但它只能是一維陣列。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將引數順位變更為 1。
