# 裝置無法使用
您的網路連線已遺失，或是您嘗試存取的裝置離線或不存在。  
  
### 更正這個錯誤  
  
1.  請檢查裝置電源和連接電腦與裝置的任何纜線。 如果您想要透過網路存取印表機，請確定您的電腦與印表機之間有邏輯連接，例如，將 LPT1 與網路印表機識別碼相關聯的連接。  
  
2.  重新連線到網路，然後再試一次。
