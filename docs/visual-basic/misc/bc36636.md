# &#39;AddressOf&#39; 運算式在 &#39;Select Case&#39; 陳述式的第一個運算式中無效
您不能使用 `AddressOf` 運算式來測試 `Select Case` 陳述式中的運算式。`AddressOf` 運算式會傳回函式委派，因此 `Select Case` 陳述式的測試運算式必須是基本資料類型。  
  
 **錯誤 ID︰**BC36636  
  
### 更正這個錯誤  
  
-   檢查您的程式碼，以判斷不同的條件式建構 \(例如 `If...Then...Else` 陳述式\) 是否可行。
