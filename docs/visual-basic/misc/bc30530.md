# 參數名稱不可與所在的函式名稱相同
您在參數及其定義函式中使用了相同的名稱。  
  
 **錯誤 ID︰**BC30530  
  
### 更正這個錯誤  
  
-   檢查函式中的參數名稱，並變更衝突的參數名稱。
