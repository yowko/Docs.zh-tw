# <a name="property-propertyname-cannot-be-initialized-in-an-object-initializer-expression-because-all-accessible-overloads-require-arguments"></a><span data-ttu-id="04b31-102">因為所有可存取的多載都需要引數，所以無法在物件初始化運算式運算式中初始化屬性 ' @no__t 0propertyname > '</span><span class="sxs-lookup"><span data-stu-id="04b31-102">Property '\<propertyname>' cannot be initialized in an object initializer expression because all accessible overloads require arguments</span></span>
<span data-ttu-id="04b31-103">物件初始設定式清單中所初始化的成員必須是欄位或屬性。</span><span class="sxs-lookup"><span data-stu-id="04b31-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="04b31-104">此外，初始設定式清單中的屬性不能有參數。</span><span class="sxs-lookup"><span data-stu-id="04b31-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="04b31-105">造成多載這個錯誤的屬性，而且它的每個版本都需要引數。</span><span class="sxs-lookup"><span data-stu-id="04b31-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="04b31-106">因此，無法初始化物件初始設定式清單中的屬性。</span><span class="sxs-lookup"><span data-stu-id="04b31-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>  
  
 <span data-ttu-id="04b31-107">**錯誤識別碼：** BC30993</span><span class="sxs-lookup"><span data-stu-id="04b31-107">**Error ID:** BC30993</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="04b31-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="04b31-108">To correct this error</span></span>  
  
- <span data-ttu-id="04b31-109">請從初始設定式清單中移除需要引數的屬性。</span><span class="sxs-lookup"><span data-stu-id="04b31-109">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="04b31-110">範例</span><span class="sxs-lookup"><span data-stu-id="04b31-110">Example</span></span>  
 <span data-ttu-id="04b31-111">下列類別包含三種屬性定義：一個用於 `TotalItems` ，兩個用於 `Item`(已多載)。</span><span class="sxs-lookup"><span data-stu-id="04b31-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```vb  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="04b31-112">`TotalItems` 屬性不需要引數，而且可以在物件初始設定清單中進行初始化 (如下列宣告所示)。</span><span class="sxs-lookup"><span data-stu-id="04b31-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="04b31-113">`Item` 屬性已多載 ，而且每個多載都需要引數。</span><span class="sxs-lookup"><span data-stu-id="04b31-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="04b31-114">因此， `Item` 不能出現在物件初始設定式清單中。</span><span class="sxs-lookup"><span data-stu-id="04b31-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```vb  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="04b31-115">若要避免這個錯誤，請在物件初始設定式外部初始化 `Item` 屬性。</span><span class="sxs-lookup"><span data-stu-id="04b31-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
