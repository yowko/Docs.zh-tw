# &#39;&lt;實作子句&gt;&#39; 無法實作 &#39;&lt;類型名稱&gt;&#39;，因為 &#39;&lt;類型名稱&gt;&#39; 是保留的名稱
`Implements` 陳述式指定了保留的名稱。  
  
 **錯誤 ID：**BC31415  
  
### 更正這個錯誤  
  
-   重新命名無法實作的介面成員。
