# XML 註解類型參數 &lt;參數&gt; 與對應的 &lt;關鍵字&gt; 陳述式上的類型參數不相符
XML 註解參數 \<參數\> 與對應的 \<關鍵字\> 陳述式上的類型參數不相符 將會忽略 XML 註解。  
  
 已針對方法的部分類別參數，但不是全部類別參數，指定方法的 XML 註解標記。  
  
 **錯誤 ID︰**BC42317  
  
### 更正這個錯誤  
  
-   請為方法中的所有類型參數提供 XML 註解標記。
