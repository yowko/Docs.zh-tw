# &#39;End Get&#39; 之前必須搭配相對應的 &#39;Get&#39;
`End Get` 已用來終止 `Get` 屬性程序。 在 `Get` 屬性程序外遇到 `End Get` 建構。  
  
 **錯誤 ID：**BC30630  
  
### 更正這個錯誤  
  
1.  請確定 `Get` 屬性程序的宣告在 `Property` 關鍵字之後，且在 `End Property` 建構之前。  
  
2.  請確定 `Get` 屬性程序的開頭為 `Get` 關鍵字，且結尾為 `End Get` 建構。
