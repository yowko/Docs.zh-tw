# 找不到委派實作所需的類型 &#39;&lt;類型名稱&gt;&#39; 之委派建構函式
包含類型的組件可能已損毀或無效。  
  
 **錯誤 ID︰**BC31074  
  
### 更正這個錯誤  
  
1.  請確認包含這種類型的組件有效。  
  
2.  再次編譯程式，看看錯誤是否重複發生。  
  
3.  如果錯誤繼續發生，請儲存您的工作，然後重新啟動 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]。
