# &#39;ReadOnly&#39; 屬性必須提供 &#39;Get&#39;
如果屬性宣告為 `ReadOnly`，則必須提供讀取其值的程序。  
  
 **錯誤 ID︰**BC30126  
  
### 更正這個錯誤  
  
1.  確定您在 `Property` 陳述式和 `End Property` 陳述式之間包含 `Get` 程序。  
  
2.  請確認已正確地終止 `Property` 宣告內的其他程序。
