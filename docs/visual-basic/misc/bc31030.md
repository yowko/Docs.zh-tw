# 專案層級的條件式編譯常數 &#39;&lt;常數名稱&gt;&#39; 無效：&lt;錯誤&gt;
專案指定了無效的條件式編譯選項。  
  
 **錯誤 ID：**BC31030  
  
### 更正這個錯誤  
  
1.  請確認與這個專案搭配使用的條件式編譯常數。
