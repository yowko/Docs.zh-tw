# 方法不可以同時有 ParamArray 和 Optional 引數
`ParamArray` 引數自動為選擇性的，而且它必須是程序宣告中的唯一選擇性引數。 所有先前的引數都必須是必要的。  
  
 **錯誤 ID︰**BC30046  
  
### 更正這個錯誤  
  
-   移除宣告中的選擇性引數。
