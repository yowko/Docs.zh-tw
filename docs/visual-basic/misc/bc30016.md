# 方法/多行 Lambda 外的標籤無效
您只能在 `Sub`、`Function`、屬性 `Get` 或屬性 `Set` 程序內將標籤加入陳述式。 只有可執行的陳述式可以有標籤，且所有可執行的陳述式都必須在程序內。  
  
 **錯誤 ID︰**BC30016  
  
### 更正這個錯誤  
  
1.  請移除陳述式中的標籤，或將陳述式移到程序內。
