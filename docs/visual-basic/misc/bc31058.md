# 和在 &lt;類型&gt; &#39;&lt;名稱3&gt;&#39; 中針對 &#39;&lt;名稱2&gt;&#39; 隱含宣告的 &#39;&lt;名稱1&gt;&#39; 產生衝突
類型成員名稱與另一個隱含建立的成員名稱相衝突。  
  
 **錯誤 ID︰**BC31058  
  
### 更正這個錯誤  
  
-   請重新命名明確宣告的成員，以移除命名衝突。
