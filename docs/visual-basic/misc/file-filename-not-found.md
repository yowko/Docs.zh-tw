# <a name="file-filename-not-found"></a>檔案 '\<檔案名稱 >' 找不到
檔案名稱不正確，或檔案不存在。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 確定檔案名稱正確且存在。
