# 陣列的引數 &#39;&lt;引數名稱&gt;&#39; 無效
陣列包含無效的引數。  
  
### 更正這個錯誤  
  
-   請檢查運算式中的引數拼法。 拼錯的變數名稱可能會隱含地建立初始化為零的數值變數。
