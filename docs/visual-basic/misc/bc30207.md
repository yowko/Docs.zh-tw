# <a name="39option-compare39-must-be-followed-by-39text39-or-39binary39"></a>&#39; option Compare 為 &#39;後面必須接著 &#39;Text &#39;或 &#39;二進位 &#39;
`Option Compare` 陳述式包含不正確的設定或沒有設定。 `Option Compare` 中唯一允許的設定是 `Text` 和 `Binary`。  
  
 **錯誤 ID︰** BC30207  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請檢查設定規範的拼字是否錯誤。  
  
2.  請將 `Text` 或 `Binary` 關鍵字加入 `Option Compare` 陳述式中，例如 `Option Compare Text`。
