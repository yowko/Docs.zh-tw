# <a name="module-statement-must-end-with-a-matching-end-module"></a>'Module' 陳述式之後必須搭配相對應的 'End Module'
模組區塊的結尾必須是 `End Module` 陳述式。  
  
 **錯誤 ID:** BC30625  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   將 `End Module` 陳述式加入 `Module` 區塊的結尾。
