# &#39;&lt;事件名稱&gt;&#39; 隱含定義 &#39;&lt;成員名稱&gt;&#39;，其與 &lt;類型&gt; &#39;&lt;類型名稱&gt;&#39; 中隱含宣告的成員互相衝突
類型成員名稱與為事件隱含建立的成員名稱相衝突。 事件會隱含建立數個變數。 例如，宣告 `Event X` 會隱含宣告名稱 `XEventHandler`、`XEvent`、`add_X` 和 `remove_X`。  
  
 **錯誤 ID︰**BC31059  
  
### 更正這個錯誤  
  
-   請重新命名明確宣告的成員，以移除命名衝突。
