# 轉換運算子無法轉換為介面類型
轉換運算子宣告時，介面類型為傳回類型。  
  
 在編譯時期，[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 會視為存在有從任何介面類型到任何介面的預先定義轉換。 這類轉換可能會在執行階段失敗，但編譯器無法預測執行階段結果，因此它可以編譯任何這類轉換。  
  
 因為編譯器認為已定義這項轉換，所以不允許您重新定義它。  
  
 **錯誤 ID︰**BC33025  
  
### 更正這個錯誤  
  
-   請完整移除這個運算子定義。 它已預先定義。
