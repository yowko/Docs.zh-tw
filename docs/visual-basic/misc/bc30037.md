# <a name="character-is-not-valid"></a>字元無效
目前的內容不允許這個字元。 例如，波狀符號 (`~`) 或分隔號 (`|`) 不可以是所宣告之項目名稱的一部分。  
  
 **錯誤 ID:** BC30037  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 檢查是否有額外的字元不小心插入程式設計項目中。  
  
2. 確認程式設計項目只使用對該項目類型有效的字元來拼寫或建構。
