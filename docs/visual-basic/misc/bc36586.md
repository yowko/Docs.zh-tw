# 未指定擴充方法 &#39;&lt;方法名稱&gt;&#39; (定義於 &#39;&lt;類型名稱&gt;&#39; 中) 的參數 &#39;&lt;參數名稱&gt;&#39; 的引數

 **錯誤 ID︰**BC36586  
  
### 更正這個錯誤  
  
-   請為指定的參數提供引數。
