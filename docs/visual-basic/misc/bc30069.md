# For 迴圈控制變數 &#39;&lt;變數名稱&gt;&#39; 已被封閉式 For 迴圈所使用
巢狀 `For` 迴圈都必須使用唯一的控制變數。  
  
 **錯誤 ID︰**BC30069  
  
### 更正這個錯誤  
  
-   變更其中一個控制變數名稱。
