# &#39;Set&#39; 方法不可以有多個參數
為屬性 `Set` 程序定義了多個參數。  
  
 **錯誤 ID︰**BC31063  
  
### 更正這個錯誤  
  
-   請從屬性 `Set` 程序移除任何多餘的參數。
