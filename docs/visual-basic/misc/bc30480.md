# 共用的 &#39;Sub New&#39; 不可以宣告為 &#39;&lt;規範&gt;&#39;
您已嘗試使用 `Shared Sub New` 宣告中的規範，這是無效的作業。  
  
 **錯誤 ID︰**BC30480  
  
### 更正這個錯誤  
  
-   請移除規範。
