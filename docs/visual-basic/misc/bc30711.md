# 方法若不在堆疊最上方，將無法設定其區域變數值
您只能修改呼叫堆疊最上方的變數。 例如，如果 `procedure1` 呼叫 `procedure2`，而您在 `procedure1` 中，則無法修改 `procedure2` 中的變數。  
  
 **錯誤 ID︰**BC30711  
  
### 更正這個錯誤  
  
-   修改呼叫堆疊最上方的變數。
