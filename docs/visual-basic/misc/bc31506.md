# 無法將 &#39;&lt;類型名稱&gt;&#39; 當做屬性使用，因為它宣告為 &#39;MustInherit&#39;
自訂屬性類別不能宣告為 `MustInherit`。  
  
 **錯誤 ID︰**BC31506  
  
### 更正這個錯誤  
  
-   請從自訂屬性移除 `MustInherit` 修飾詞。
