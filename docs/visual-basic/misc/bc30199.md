# 必須有 &#39;(&#39;
程序呼叫中的引數清單開頭不是左括弧 \(`(`\) 字元。  
  
 **錯誤 ID︰**BC30199  
  
### 更正這個錯誤  
  
-   請在清單開頭加入左括弧字元。
