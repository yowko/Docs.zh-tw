# 宣告為 &#39;ReadOnly&#39; 的屬性不能有 &#39;Set&#39;
`Set` 程序會寫入屬性的值。 無法寫入 `ReadOnly` 屬性。  
  
 **錯誤 ID︰**BC30022  
  
### 更正這個錯誤  
  
-   請從 `Property` 陳述式中移除 `ReadOnly` 關鍵字，或從屬性主體中移除 `Set` 程序。
