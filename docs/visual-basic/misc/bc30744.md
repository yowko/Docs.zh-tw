# 所參考陣列項目的值為 &#39;Nothing&#39;
尚未設定所參考的陣列項目，或先前設定為 `Nothing`。  
  
 **錯誤 ID︰**BC30744  
  
### 更正這個錯誤  
  
-   請確認已正確地設定陣列中的值。
