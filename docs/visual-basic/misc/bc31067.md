# 結構中的方法不可宣告為 &#39;Protected&#39; 或 &#39;Protected Friend&#39;
結構中的方法已宣告為 `Protected` 或 `Protected Friend`。  
  
 **錯誤 ID︰**BC31067  
  
### 更正這個錯誤  
  
-   將結構內的方法宣告為 `Public` 或 `Private`。
