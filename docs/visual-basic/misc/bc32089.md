# <a name="ltparameternamegt-is-already-declared-as-a-type-parameter-of-this-method"></a>'&lt;parametername&gt;' 已經宣告為此方法的型別參數
泛型程序使用了與類型參數相同的名稱來定義一般參數或區域變數。  
  
 程序的每個參數 (包括泛型程序的每個類型參數) 都必須具有與所有其他參數不同的名稱。 由於程序參數會當做區域變數使用，因此程序內宣告的所有區域變數也必須具有與所有參數和類型參數不同的名稱。  
  
 **錯誤 ID:** BC32089  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   變更一般參數或區域變數的名稱。
