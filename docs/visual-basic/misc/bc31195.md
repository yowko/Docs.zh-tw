# 未定義類型 &#39;type1&#39; 和 &#39;type2&#39; 的運算子 &#39;operator&#39;
未定義類型 'type1' 和 'type2' 的運算子 'operator'。 您可以使用 'Value' 屬性來取得 '\<父項目\>' 的第一個項目的字串值。  
  
 二元或 `Boolean` 比較中已使用 XML 常值。  
  
 **錯誤 ID︰**BC31195  
  
### 更正這個錯誤  
  
-   使用 XML 常值的 `Value` 屬性來進行二元或 `Boolean` 比較。
