# 專案 &#39;&lt;專案名稱&gt;&#39; 無法產生參考至檔案 &#39;&lt;檔案名稱&gt;&#39;
專案 '\<`projectname`\>' 無法產生參考至檔案 '\<`filename`\>'。 您必須刪除並重新加入此參考，然後完整重建這個方案。  
  
 建立檔案的參考時發生錯誤。  
  
 **錯誤 ID︰**BC31514  
  
### 更正這個錯誤  
  
1.  移除參考，方法是在**方案總管**的 \[專案參考\] 中，以滑鼠右鍵按一下項目，然後按一下 \[移除\]。  
  
2.  還原參考，方法是在**方案總管**中，以滑鼠右鍵按一下 \[參考\]，然後按一下 \[加入參考\]。
