# 全形字元不可以當做 XML 分隔符號
XML 常值已定義，其中包含作為分隔符號的全形字元。 全形字元也稱為寬字元或多位元組字元。  
  
 **錯誤 ID︰**BC31197  
  
### 更正這個錯誤  
  
-   從 XML 常值定義中移除全形字元，並取代為有效的 ANSI 分隔符號字元。 有效的分隔符號字元包括：`<`、`>`、`=`、`:`、`/`。
