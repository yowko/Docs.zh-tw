# 委派無法處理事件
委派是一種參考類型，指向共用程序或指向物件上的執行個體。 因為透過指派可以變更它所指向的程序，所以 `Delegate` 陳述式不支援 `Handles` 或 `Implements` 子句。  
  
 **錯誤 ID︰**BC30019  
  
### 更正這個錯誤  
  
-   請從 `Delegate` 陳述式中移除 `Handles` 子句。
