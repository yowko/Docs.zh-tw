# &#39;Namespace&#39; 陳述式之後必須搭配相對應的 &#39;End Namespace&#39;
`Namespace` 區塊的結尾必須是 `End Namespace` 陳述式。  
  
 **錯誤 ID：**BC30626  
  
### 更正這個錯誤  
  
-   將 `End Namespace` 陳述式加入 `Namespace` 區塊的結尾。
