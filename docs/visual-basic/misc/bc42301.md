# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>每個語言項目只允許一個 XML 註解區塊
已將多個 XML 註解區塊套用至某個語言項目。  
  
 **錯誤 ID：** BC42301  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請移除多餘的 XML 註解區塊。
