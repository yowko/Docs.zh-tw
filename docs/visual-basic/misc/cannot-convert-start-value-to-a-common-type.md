# 無法將類型 &#39;&lt;類型1&gt;&#39; 的起始值、&#39;&lt;類型2&gt;&#39; 的限制值和 &#39;&lt;類型3&gt;&#39; 的間距值轉換為一般類型
指定的 `For...Next` 控制項值具有不同的類型，因此無效，因為它們無法轉換成一般類型。  
  
### 更正這個錯誤  
  
1.  變更一個或多個值的類型，以全部轉換成一般類型。
