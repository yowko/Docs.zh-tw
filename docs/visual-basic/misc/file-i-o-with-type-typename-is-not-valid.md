# 具有類型 &#39;&lt;類型名稱&gt;&#39; 的檔案 I/O 是無效的
您正在嘗試將不支援的類型寫入檔案。  
  
### 更正這個錯誤  
  
-   請檢查結構以決定不受支援類型的發生位置，然後將它變更為有效的類型。
