# <a name="extension-method-methodname-defined-in-typename-does-not-have-the-same-signature-as-delegate-delegatename"></a><span data-ttu-id="ab816-102">擴充方法 '\<方法名稱 >' 中定義'\<類型名稱 >' 沒有相同的簽章與委派 '\<委派名稱 >'</span><span class="sxs-lookup"><span data-stu-id="ab816-102">Extension method '\<methodName>' defined in '\<typeName>' does not have the same signature as delegate '\<delegateName>'</span></span>
<span data-ttu-id="ab816-103">您嘗試使用之擴充方法和委派的簽章不符。</span><span class="sxs-lookup"><span data-stu-id="ab816-103">There is a mismatch between the signatures of the extension method and the delegate that you are attempting to use.</span></span> <span data-ttu-id="ab816-104">`Delegate` 陳述式定義委派類別的參數類型和傳回類型。</span><span class="sxs-lookup"><span data-stu-id="ab816-104">The `Delegate` statement defines the parameter types and return types of a delegate class.</span></span> <span data-ttu-id="ab816-105">任何具有相符參數、類型和傳回類型的程序都可用來建立這個委派類型的執行個體。</span><span class="sxs-lookup"><span data-stu-id="ab816-105">Any procedure with matching parameters, types, and return types can be used to create an instance of this delegate type.</span></span> <span data-ttu-id="ab816-106">下列範例會報告這個錯誤，因為擴充方法的簽章 `Example` 與委派簽章 `Del`不相容。</span><span class="sxs-lookup"><span data-stu-id="ab816-106">This error is reported in the following example because the signature of extension method `Example` is not compatible with the signature of delegate `Del`.</span></span>  
  
```vb  
' Definition of the delegate, with two parameters.  
Delegate Sub Del(ByVal m As Integer, ByVal s As String)  
```  
  
```vb  
' Definition of the extension method, with one parameter.  
<Extension()> _  
Sub Example(ByVal s As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
'' This assignment causes the error.  
' Dim exampleDel As Del = AddressOf Example  
```  
  
 <span data-ttu-id="ab816-107">**錯誤 ID:** BC36580</span><span class="sxs-lookup"><span data-stu-id="ab816-107">**Error ID:** BC36580</span></span>

- <span data-ttu-id="ab816-109">確認委派和擴充方法的參數數目相同。</span><span class="sxs-lookup"><span data-stu-id="ab816-109">Verify that the delegate and the extension method have the same number of parameters.</span></span>

- <span data-ttu-id="ab816-111">比較每個委派參數的資料類型與對應擴充方法參數的資料類型，以確定它們相容。</span><span class="sxs-lookup"><span data-stu-id="ab816-111">Compare the data type of each delegate parameter to the data type of the corresponding extension method parameter to make sure they are compatible.</span></span>
