# 設定組件資訊清單選項時發生錯誤: &lt;錯誤訊息&gt;
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 編譯器呼叫組件連結器 \(Al.exe，也稱為 Alink\) 使用資訊清單產生組件。 尋找用來簽署組件的金鑰檔或金鑰名稱時，連結器回報錯誤。  
  
 **錯誤 ID︰**BC30139  
  
### 更正這個錯誤  
  
1.  檢查引用的錯誤訊息，並參考名為 [Al.exe 工具錯誤和警告](http://msdn.microsoft.com/zh-tw/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)的頁面，以取得進一步說明和建議。  
  
2.  如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
