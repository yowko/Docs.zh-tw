# 無法在介面中宣告 &#39;Sub New&#39;
您已嘗試在介面內宣告 `Sub New`。 因為它是建構函式，所以 `Sub New` 只能在建立類別時執行一次。 不能在相同類別或衍生類別的其他建構函式的第一行程式碼以外的任何地方明確呼叫它。  
  
 **錯誤 ID︰**BC30363  
  
### 更正這個錯誤  
  
-   請移除 `Sub New`，或將它移至您程式碼中的更適當位置。
