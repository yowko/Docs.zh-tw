# XML 註解必須是該行的第一個陳述式
XML 註解必須是該行的第一個陳述式。 將會忽略 XML 註解。  
  
 XML 註解被放在另一個語言項目之後，這是不允許的做法。  
  
 **錯誤 ID︰**BC42302  
  
### 更正這個錯誤  
  
-   將 XML 註解移至新的一行。
