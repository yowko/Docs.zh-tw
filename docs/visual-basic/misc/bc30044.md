# &#39;&lt;關鍵字&gt;&#39; 在結構中無效
結構是實值類型，而不是參考類型。 結構不是從類別建立的執行個體，因此 `Me`、`MyClass` 和 `MyBase` 關鍵字在結構中沒有意義。  
  
 **錯誤 ID︰**BC30044  
  
### 更正這個錯誤  
  
-   請將結構變更為類別，或從程序中移除關鍵字。
