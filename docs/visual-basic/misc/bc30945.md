# &#39;&lt;類型名稱&gt;&#39; 類型無法實作介面 &#39;&lt;介面名稱&gt;&#39;，因為它宣告具有傳回類型的 &lt;事件簽章&gt;
類別或結構嘗試實作的介面，宣告會傳回值的事件。  
  
 Visual Basic 目前不支援宣告會傳回值的事件。  
  
 **錯誤 ID︰**BC30945  
  
### 更正這個錯誤  
  
-   請從類別或結構定義移除 `Implements` 陳述式，或實作不同的介面。
