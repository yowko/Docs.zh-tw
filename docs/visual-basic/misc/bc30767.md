# 已不再支援 &#39;Get&#39; 陳述式 (Visual Basic)
不再支援 `Get` 陳述式。 檔案 I\/O 功能通常是在 `Microsoft.VisualBasic` 命名空間中，但是目標 .NET Compact Framework 版本並不支援它。  
  
 **錯誤 ID：**BC30767  
  
### 更正這個錯誤  
  
-   使用 <xref:System.IO> 命名空間中所定義的函式來執行檔案作業。
