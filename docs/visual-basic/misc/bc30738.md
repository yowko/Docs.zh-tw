# &#39;Sub Main&#39; 在 &#39;&lt;命名空間&gt;&#39; 中已經重複宣告: &lt;訊息&gt;
`Sub Main` 只能在命名空間內宣告一次。  
  
 **錯誤 ID︰**BC30738  
  
### 更正這個錯誤  
  
-   確定您的專案中只有一個 `Sub Main` 程序。
