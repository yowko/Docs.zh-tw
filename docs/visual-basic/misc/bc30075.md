# 具名引數不可以當做陣列註標
使用依名稱傳遞程序引數的語法，來指定陣列索引 \(例如 `Array(Index := 10)`\)。 這個語法不適用於陣列註標。  
  
 **錯誤 ID︰**BC30075  
  
### 更正這個錯誤  
  
-   使用一般變數或常數運算式作為陣列索引 \(例如 `Array(10)`\)。
