# 宣告為 &#39;Overrides&#39; 的方法無法宣告為 &#39;Overridable&#39;，因為它們可以被隱含地覆寫
與大多數的方法不同，標記為 `Overrides` 修飾詞的方法預設是可覆寫的。  
  
 **錯誤 ID︰**BC30730  
  
### 更正這個錯誤  
  
-   省略標記為 `Overrides` 修飾詞之方法中的 `Overridable` 修飾詞。
