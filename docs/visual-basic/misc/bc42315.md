# &#39;declare sub&#39; 語言項目上不可使用 XML 註解標記 &#39;returns&#39;
'declare sub' 語言元素上不可使用 XML 註解標記 'returns'。 將會忽略 XML 註解。  
  
 `Declare Sub` 宣告的 XML 註解不能包含 `<`returns`>` 標記。  
  
 **錯誤 ID︰**BC42315  
  
### 更正這個錯誤  
  
-   請移除不支援的 `<`returns`>` 標記。
