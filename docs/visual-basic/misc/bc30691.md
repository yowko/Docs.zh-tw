# &#39;&lt;類型名稱&gt;&#39; 是 &#39;&lt;容器名稱&gt;&#39; 的類型，不能當成運算式使用
正在使用的名稱之一已經宣告為類型名稱，不能用在運算式中。  
  
 **錯誤 ID：**BC30691  
  
### 更正這個錯誤  
  
-   成員的唯一名稱。  
  
-   成員名稱請使用方括弧括住。
