# <a name="property-propertyname-cannot-be-initialized-in-an-object-initializer-expression-because-it-requires-arguments"></a>屬性 ' \<propertyname > ' 無法在物件初始化運算式運算式中初始化，因為它需要引數
物件初始設定式清單中所初始化的成員必須是欄位或屬性，而且屬性成員不能有參數。 例如，預設屬性需要有引數，因此不能在物件初始設定式清單中進行初始化。 如需詳細資訊，請參閱 。  
  
 **錯誤識別碼：** BC30992  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從初始設定清單中移除具有引數的所有屬性。  
  
## <a name="example"></a>範例  
 下列類別包含需要整數引數的預設屬性 `defaultProp`。  
  
```vb  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```  
  
 因為預設屬性需要有引數，所以下列宣告會造成錯誤。  
  
```vb  
' Dim strs As New SomeStrings(2) With { .defaultProp = "One" }  
```
