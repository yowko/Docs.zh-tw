# &#39;&lt;rsetstmt&gt;&#39; 未宣告
'\<rsetstmt\>' 未宣告。 不再支援 RSet 陳述式。請使用 Microsoft.VisualBasic.RSet 代替。  
  
 舊版 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 中內建的幾個函式已移至 <xref:Microsoft.VisualBasic?displayProperty=fullName> 命名空間。 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰**BC30821  
  
### 更正這個錯誤  
  
-   改為在 `Microsoft.VisualBasic` 命名空間中使用 `RSet` 函式。
