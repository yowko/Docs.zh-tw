# 無法將 &#39;&lt;類型名稱&gt;&#39; 當做屬性使用，因為它不是繼承自 &#39;System.Attribute&#39;
嘗試使用不是繼承自 `System.Attribute` 的類別。  
  
 **錯誤 ID︰**BC31504  
  
### 更正這個錯誤  
  
1.  請定義自訂屬性作為衍生自 `System.Attribute` 的類別，方法是將 `Imports` 陳述式加入類別中的第一行程式碼。
