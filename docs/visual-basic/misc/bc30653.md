# 需要模組 &#39;&lt;模組名稱&gt;&#39; (包含類型 &#39;&lt;成員名稱&gt;&#39;) 的參考
需要模組 '\<模組名稱\>' \(包含類型 '\<成員名稱\>'\) 的參考。 請在專案中加入一個參考。  
  
 專案所含的類型取決於未包含在專案參考之組件的另一個類型。  
  
 **錯誤 ID︰**BC30653  
  
### 更正這個錯誤  
  
-   請在專案中加入相依類型的專案參考。
