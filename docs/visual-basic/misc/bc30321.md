# 在相同的 &#39;Select&#39; 陳述式中，&#39;Case&#39; 不可以跟隨在 &#39;Case Else&#39; 之後
如果沒有找到初始 `Case` 的相符項，`Case Else` 陳述式導致執行陳述式。 在同一個 `Select` 區塊的 `Case Else` 之後找到 `Case` 陳述式。  
  
 **錯誤 ID︰**BC30321  
  
### 更正這個錯誤  
  
-   請將 `Case Else` 移至 `Case` 陳述式之後的適當位置。
