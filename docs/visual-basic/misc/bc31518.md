# 無法建立組件 &#39;&lt;組件名稱&gt;&#39;，因為路徑超過 259 個字元
這個組件的路徑超過 259 個字元的長度上限。  
  
 **錯誤 ID︰**BC31518  
  
### 更正這個錯誤  
  
-   使用較短的名稱，或選取較淺的檔案階層，來建立較短的路徑。
