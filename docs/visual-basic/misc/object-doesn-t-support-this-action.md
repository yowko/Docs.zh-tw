# 物件不支援此動作 (Visual Basic)
您已參考這個物件不支援的方法或屬性。  
  
### 更正這個錯誤  
  
1.  如需物件的詳細資訊，請閱讀物件的文件。  
  
2.  請確定屬性和方法的名稱拼寫正確。
