# 類型字元 &#39;&lt;字元&gt;&#39; 不可以使用在明確類型的宣告中
明確指定類型時使用了類型字元，例如 `$`。  
  
 **錯誤 ID︰**BC30302  
  
### 更正這個錯誤  
  
-   請從宣告移除類型字元，或移除 `As <Type>` 子句。
