# &#39;#End Region&#39; 之前必須搭配相對應的 &#39;#Region&#39;
透過 `#Region`，您可以在使用 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] 程式碼編輯器的大綱功能時，指定要展開或摺疊程式碼區塊。`#Region` 陳述式的開頭和結尾必須在相同的程式碼區塊中。  
  
 **錯誤 ID：**BC30680  
  
### 更正這個錯誤  
  
-   請在對應 `#End``Region` 陳述式之前的適當位置插入 `#Region`。
