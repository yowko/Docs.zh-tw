# 無法在 Lambda 運算式內使用 GoTo 標籤 &#39;&lt;標籤名稱&gt;&#39;
Lambda 運算式的主體必須是計算並傳回單一值的運算式。 不允許任何陳述式，包括 `GoTo` 陳述式。  
  
 **錯誤 ID︰**BC36597
