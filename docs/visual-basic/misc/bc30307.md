# &#39;&lt;方法1&gt;&#39; 無法覆寫 &#39;&lt;方法2&gt;&#39;，因為其選擇性參數的預設值不同
您嘗試以另一個方法覆寫某個方法，但其選擇性參數的預設值與後者不同，這表示兩者的簽章不同。 類型可能會藉由宣告具有相同名稱和特徵標記的方法，並使用 `Overrides` 修飾詞標記宣告，來覆寫繼承的可覆寫方法。  
  
 **錯誤 ID︰**BC30307  
  
### 更正這個錯誤  
  
-   確定兩個方法具有相同的特徵標記。
