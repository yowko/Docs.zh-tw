# <a name="method1-cannot-override-method2-because-they-differ-by-the-default-values-of-optional-parameters"></a>'\<method1 > ' 無法覆寫\<' method2 > ', 因為它們的預設值與選擇性參數不同
您嘗試以另一個方法覆寫某個方法，但其選擇性參數的預設值與後者不同，這表示兩者的簽章不同。 類型可能會藉由宣告具有相同名稱和特徵標記的方法，並使用 `Overrides` 修飾詞標記宣告，來覆寫繼承的可覆寫方法。  
  
 **錯誤識別碼:** BC30307  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確定兩個方法具有相同的特徵標記。
