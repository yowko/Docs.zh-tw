# <a name="paramarray-parameters-must-be-declared-39byval39"></a>ParamArray 參數必須宣告為 &#39;ByVal &#39;
`ParamArray` 參數不能使用 `ByRef` 修飾詞。  
  
 **錯誤識別碼：** BC30667  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `ParamArray` 修飾詞宣告 `ByVal` 。
