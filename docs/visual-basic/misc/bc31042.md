# &#39;Sub New&#39; 無法實作介面成員
`Sub New` 是建構函式，而且無法實作成員。  
  
 **錯誤 ID︰**BC31042  
  
### 更正這個錯誤  
  
-   從 `Sub New` 程序中移除 `Implements` 陳述式。
