# 無法使用 &lt;數字&gt; 個引數呼叫方法 &#39;&lt;方法名稱&gt;&#39;
方法呼叫包含錯誤的引數數目。  
  
### 更正這個錯誤  
  
1.  請計算您提供的引數數目。  
  
2.  比對這個數目與您想要呼叫之方法中的參數數目。  
  
3.  如果這兩個數目不一致，請調整引數數目。
