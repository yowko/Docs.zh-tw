# 必須要有逗號或 &#39;)&#39; (類型參數清單)
類型參數清單包含未預期的項目。 請使用逗號 \(`,`\) 來分隔這類清單中的類型參數，並使用右括弧 \(`)`\) 來結束清單。  
  
 **錯誤 ID︰**BC32099  
  
### 更正這個錯誤  
  
-   請加入逗號並繼續類型參數清單，或者加入右括弧字元來終止清單。
