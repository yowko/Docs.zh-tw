# 設計工具產生的類型 &#39;&lt;類型&gt;&#39; 中的 &#39;&lt;建構函式&gt;&#39;，應該呼叫 InitializeComponent 方法
設計工具產生之類型中的建構函式未呼叫類型的 `InitializeComponent` 方法。  
  
 設計工具產生之類型中的每個建構函式都應該呼叫類型的 `InitializeComponent` 方法。  
  
 **錯誤 ID︰**BC40054  
  
### 更正這個錯誤  
  
-   請在建構函式中加入 `InitializeComponent` 方法的呼叫。
