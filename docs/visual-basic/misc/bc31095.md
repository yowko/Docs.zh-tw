# 呼叫其他建構函式時，參考建構中的物件是無效的
在物件的建構函式建立好物件之前，已建立對該物件成員的參考。  
  
 **錯誤 ID：**BC31095  
  
### 更正這個錯誤  
  
-   從另一個建構函式呼叫建構函式時，請勿使用 `MyBase`、`MyClass` 或 `Me`。
