# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39;EndIf&#39;不再支援陳述式。使用&#39;End If&#39;改為
不再支援 `EndIf` 陳述式。  
  
 **錯誤 ID︰** BC30826  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `End If` 來終止 `If` 陳述式。
