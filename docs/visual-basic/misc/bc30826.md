# 不再支援 &#39;EndIf&#39; 陳述式; 請使用 &#39;End If&#39; 代替
不再支援 `EndIf` 陳述式。  
  
 **錯誤 ID︰**BC30826  
  
### 更正這個錯誤  
  
-   使用 `End If` 來終止 `If` 陳述式。
