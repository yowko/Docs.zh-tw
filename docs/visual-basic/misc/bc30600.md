# <a name="39withevents39-variable-does-not-raise-any-instance-events-that-are-accessible-to-39ltcontainernamegt39"></a>&#39;WithEvents&#39;變數沒有引發任何可存取的執行個體事件&#39; &lt;containername&gt;&#39;
您所參考的物件未引發可存取的事件。  
  
 **錯誤 ID︰** BC30600  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  修改事件的存取修飾詞，或選擇可在目前內容中存取的事件。
