# 從 &#39;Double&#39; 轉換成 &#39;Date&#39; 需要呼叫 &#39;Date.FromOADate&#39;
您嘗試將 `Date` 值轉換成 `Double` 值，若不使用 <xref:System.DateTime.FromOADate%2A?displayProperty=fullName> 方法便無法完成。  
  
 **錯誤 ID：**BC30533  
  
### 更正這個錯誤  
  
-   請使用 <xref:System.DateTime.FromOADate%2A> 方法來轉換值。
