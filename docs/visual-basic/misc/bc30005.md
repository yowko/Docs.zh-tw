# 組件 &#39;&lt;組件名稱&gt;&#39; (包含事件 &#39;&lt;事件名稱&gt;&#39; 的定義) 需要參考
組件 '\<`assemblyname`\>' \(包含事件 '\<`eventname`\>' 的定義\) 需要參考。 請將參考加入您的專案中。  
  
 此事件是在專案中未直接參考的動態連結程式庫 \(DLL\) 或組件中所定義。[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 編譯器需要參考，以避免當事件在多個 DLL 或組件中定義時所發生的模稜兩可情況。  
  
 **錯誤 ID︰**BC30005  
  
### 更正這個錯誤  
  
-   在您的專案參考中包含未參考之 DLL 或組件的名稱。
