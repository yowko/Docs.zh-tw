# 類型 &#39;&lt;類型名稱&gt;&#39; 未定義或包含它的模組未於偵錯工作階段中載入
參考的類型無法使用或未定義。  
  
 **錯誤 ID︰**BC30743  
  
### 更正這個錯誤  
  
1.  驗證類型名稱的拼寫正確。  
  
2.  定義目前類別或模組內的類型。
