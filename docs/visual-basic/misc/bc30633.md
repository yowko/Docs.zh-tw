# &#39;Set&#39; 陳述式之後必須搭配相對應的 &#39;End Set&#39;
`Set` 屬性程序的結尾必須是 `End Set` 陳述式。  
  
 **錯誤 ID：**BC30633  
  
### 更正這個錯誤  
  
-   請確定 `Set` 屬性程序的結尾為 `End Set` 建構。
