# &#39;!&#39; 的左運算元必須有類型參數、類別或介面類型，但此運算元具有類型 &#39;&lt;類型&gt;&#39;
字典成員存取 \(使用驚嘆號 \(`!`\) 而非句號 \(`.`\)\) 只有在類別或介面上有效。  
  
 **錯誤 ID︰**BC30103  
  
### 更正這個錯誤  
  
-   請將驚嘆號左邊的運算式取代為評估為已定義類別或介面類型的運算式。
