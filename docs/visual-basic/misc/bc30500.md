# 常數 &#39;&lt;常數名稱&gt;&#39; 無法相依在本身的值上
您已在程式碼中建立循環相依性，其中常數取決於它自己的值。 例如，`Const a = Const b; Const b = Const a`。  
  
 **錯誤 ID︰**BC30500  
  
### 更正這個錯誤  
  
1.  請檢查您的程式碼，以查看評估常數的位置，並相應地進行修改。
