# &#39;Finally&#39; 的分支無效
`Finally` 區塊內的 `GoTo` 陳述式分叉到區塊之外。 分叉進入或離開 `Catch` 或 `Finally` 區塊是無效的。  
  
 **錯誤 ID︰**BC30101  
  
### 更正這個錯誤  
  
-   請移除 `GoTo` 陳述式，並考慮使用決策或迴圈控制結構來實作程式邏輯。
