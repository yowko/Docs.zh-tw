# &#39;System.Runtime.InteropServices.DllImportAttribute&#39; 無法套用至 &#39;Get&#39; 或 &#39;Set&#39;
`DllImportAttribute` 屬性已套用至 `Get` 或 `Set` 屬性程序。  
  
 **錯誤 ID︰**BC31524  
  
### 更正這個錯誤  
  
1.  請從 `Get` 和 `Set` 屬性程序中移除 `DllImportAttribute`。
