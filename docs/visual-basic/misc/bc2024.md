# 已經自動加入預設組件參考 &#39;&lt;參考&gt;&#39;; 已忽略
已嘗試加入預設組件參考，但組件已經加入，因此忽略該嘗試。  
  
 **錯誤 ID︰**BC2024  
  
### 更正這個錯誤  
  
-   請移除多餘的程式碼。
