# &#39;Custom&#39; 修飾詞在以介面宣告的事件中無效
無法在介面中宣告自訂事件，因為自訂事件必須提供其 `AddHandler`、`RemoverHandler` 和 `RaiseEvent` 方法的實作。  
  
 `Custom` 關鍵字可用於實作事件的衍生類別。  
  
 **錯誤 ID︰**BC31121  
  
### 更正這個錯誤  
  
-   從介面的事件宣告中移除 `Custom` 關鍵字。
