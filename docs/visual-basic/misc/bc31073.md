# &#39;&lt;名稱1&gt;&#39; 與 &#39;&lt;名稱2&gt;&#39; 中所定義的同名 &lt;類型&gt; 衝突
發現名稱與同名的類型衝突。  
  
 **錯誤 ID︰**BC31073  
  
### 更正這個錯誤  
  
1.  宣告唯一名稱。  
  
2.  使用方括號括住衝突的成員名稱。
