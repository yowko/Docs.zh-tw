# 隱含變數 &#39;&lt;變數名稱&gt;&#39; 無效，原因為 &#39;&lt;訊息&gt;&#39;

 **錯誤 ID︰**BC36016  
  
### 更正這個錯誤  
  
1.  請檢查指出的訊息中是否有原因和可能的補救方法。  
  
2.  如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
