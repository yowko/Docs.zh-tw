# &#39;Next&#39; 控制變數與 &#39;For&#39; 迴圈控制變數不相符
`For...Next` 迴圈之 `Next` 陳述式中的控制變數必須符合對應 `For` 陳述式中的變數。  
  
 **錯誤 ID︰**BC30976  
  
### 更正這個錯誤  
  
-   檢查 `Next` 陳述式與對應 `For` 陳述式中的變數拼字，確定相符。  
  
-   確定沒有不小心刪除封閉式迴圈的任何部分。  
  
-   如果這個迴圈是一組巢狀迴圈的一部分，請確定已正確地終止每個迴圈。
