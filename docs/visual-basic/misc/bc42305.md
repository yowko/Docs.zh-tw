# 在同一個 XML 註解區塊中，具有相同屬性的 XML 註解標記 &#39;&lt;標記&gt;&#39; 出現多次
在同一個 XML 註解區塊中，具有相同屬性的 XML 註解標記 \<標記\> 出現多次。 將會忽略 XML 註解。  
  
 XML 註解包含多個具有相同屬性的標記。  
  
 **錯誤 ID︰**BC42305  
  
### 更正這個錯誤  
  
-   請移除重複的標記。
