# 無法將安全屬性 &#39;&lt;屬性名稱&gt;&#39; 套用至模組
Common Language Runtime 不接受模組上的安全屬性。  
  
 **錯誤 ID︰**BC30131  
  
### 更正這個錯誤  
  
1.  請從宣告中移除屬性。
