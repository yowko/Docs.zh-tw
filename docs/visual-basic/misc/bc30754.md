# &#39;GoTo &lt;標籤名稱&gt;&#39; 無效，因為 &#39;&lt;標籤名稱&gt;&#39; 位於不包含此陳述式的 &#39;Try&#39;、&#39;Catch&#39; 或 &#39;Finally&#39; 陳述式內部
您無法分支到 `Try...Catch...Finally` 區塊。  
  
 **錯誤 ID︰**BC30754  
  
### 更正這個錯誤  
  
-   重建您的程式碼，讓標籤位於 `Try...Catch...Finally` 區塊之前。
