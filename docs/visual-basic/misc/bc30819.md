# 未宣告 &#39;&lt;數學函式1&gt;&#39;
未宣告 '\<數學函式1\>'。 函式已經移動至 System.Math 類別，現在稱為 'mathfunction2'。  
  
 舊版 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 中內建的幾個函式已移至 <xref:System.Math?displayProperty=fullName> 命名空間。 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰**BC30819  
  
### 更正這個錯誤  
  
-   請使用 <xref:System.Math?displayProperty=fullName> 中宣告的方法。
