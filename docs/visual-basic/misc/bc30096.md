# &#39;Exit For&#39; 只可以在 &#39;For&#39; 陳述式中出現
`Exit For` 陳述式出現在 `For` 迴圈外。`Exit For` 僅適用於 `For` 或 `For Each` 陳述式與其相對應的 `Next` 陳述式之間。  
  
 **錯誤 ID︰**BC30096  
  
### 更正這個錯誤  
  
1.  請確定有效的 `For` 或 `For Each` 陳述式出現在 `Exit For` 前面，而它的後面出現有效的 `Next` 陳述式。  
  
2.  請確認已正確地終止 `For` 迴圈內的其他控制結構。
