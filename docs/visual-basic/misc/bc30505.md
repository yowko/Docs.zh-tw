# 實作介面成員的方法或事件不可以宣告為 &#39;Shared&#39;
您已嘗試將實作介面成員的方法或事件宣告為 `Shared`。 類別中所實作的方法和事件不能指定為 `Shared` 或 `Private` \(在不可繼承的類別中除外\)。  
  
 **錯誤 ID︰**BC30505  
  
### 更正這個錯誤  
  
-   請移除 `Shared` 關鍵字。
