# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>'RemoveHandler' 宣告之後必須搭配相對應的 'End RemoveHandler'
`RemoveHandler` 宣告的結尾必須是 `End RemoveHandler` 陳述式。  
  
 **錯誤 ID:** BC31116  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請檢查 `RemoveHandler` 宣告的結尾是否為 `End RemoveHandler` 陳述式。
