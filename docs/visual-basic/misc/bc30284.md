# &lt;類型1&gt; &#39;&lt;類型名稱&gt;&#39; 不可宣告為 &#39;Overrides&#39;，因為其在基底 &lt;類型2&gt; 中不會覆寫 &lt;類型1&gt;。
`Sub`、`Function` 或 `Property` 陳述式指定 `Overrides`，但基底類別中沒有相同名稱的類型存在。  
  
 **錯誤 ID︰**BC30284  
  
### 更正這個錯誤  
  
1.  請檢查類型名稱拼寫正確。  
  
2.  請移除多餘的 `Overrides` 關鍵字。
