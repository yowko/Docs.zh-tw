# 以單一類型規範來宣告多個變數時不允許明確初始化
在同一行中宣告多個變數時，不允許初始化。  
  
 **錯誤識別碼：**BC30671  
  
### 更正這個錯誤  
  
1.  分別宣告及初始化每個項目。  
  
2.  同時宣告多個項目，再初始化每個項目，例如：  
  
    ```  
    Dim x, b, i As Integer x = 9 : b = 9 : i = 9 ' ":" is the same as a new line.  
    ```
