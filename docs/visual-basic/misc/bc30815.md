# <a name="ltfunctionnamegt-is-not-declared-visual-basic-error"></a>'&lt;functionname&gt;' 未宣告 （Visual Basic 錯誤）
'\<函數名稱 >' 未宣告。 檔案 I/O 功能可以在 Microsoft.VisualBasic 命名空間中使用。  
  
 **錯誤 ID:** BC30815  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   使用 `My.Computer.FileSystem`中所定義的檔案 I/O 函式、 `System.IO` 命名空間、Visual Basic 執行階段函式，或 `Microsoft.VisualBasic.FileSystem` 物件。
