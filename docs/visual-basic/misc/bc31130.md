# 事件 &#39;&lt;事件名稱&gt;&#39; 遺漏 &#39;AddHandler&#39; 定義
如果事件宣告為 `Custom`，則必須提供用於加入事件處理常式的程序。  
  
 **錯誤 ID︰**BC31130  
  
### 更正這個錯誤  
  
1.  在 `Custom Event` 陳述式與 `End Event` 陳述式之間加入 `AddHandler` 宣告。  
  
2.  請確認已正確地終止事件宣告內的其他程序。
