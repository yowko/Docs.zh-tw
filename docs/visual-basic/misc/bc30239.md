# 必須有關係運算子
`Case` 陳述式包含 `Is` 子句，但沒有比較運算子 \(如 `=` 或 `>`\)。 如果 `Case` 陳述式不包括運算子，則會採用 `=`，而且不會使用 `Is`。  
  
 **錯誤 ID︰**BC30239  
  
### 更正這個錯誤  
  
-   請移除 `Is` 關鍵字，或在它的後面加上比較運算子。
