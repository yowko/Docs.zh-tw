# 找不到 .NET Framework 目錄: &lt;錯誤&gt;
找不到儲存 [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] 組件的目錄。  
  
 **錯誤 ID︰**BC31508  
  
### 更正這個錯誤  
  
1.  再次編譯程式，看看錯誤是否重複發生。  
  
2.  如果錯誤繼續發生，請儲存您的工作，然後重新啟動 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)]。  
  
3.  如果問題重複發生，請重新安裝 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)]。  
  
4.  如果錯誤在重新安裝之後持續發生，請通知 Microsoft 產品支援服務。
