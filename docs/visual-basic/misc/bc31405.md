# <a name="optional-parameters-cannot-have-structure-types"></a>選擇性參數不能有結構類型
選擇性參數不能是結構類型。  
  
 **錯誤 ID：** BC31405  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  視需要宣告結構類型參數。  
  
2.  宣告靜態類別成員取代結構。
