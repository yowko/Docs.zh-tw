# 無法將類型 &#39;&lt;類型名稱&gt;&#39; 當做屬性使用，因為它未宣告為 &#39;Public&#39;
屬性必須是公用的。  
  
 **錯誤 ID︰**BC31516  
  
### 更正這個錯誤  
  
1.  請將 `Public` 修飾詞加入屬性類別。
