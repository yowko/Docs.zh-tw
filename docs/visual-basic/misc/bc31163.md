# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>XML 屬性值的結束單引號必須相符
以單引號分隔的屬性值遺漏右引號字元。  
  
 **錯誤 ID︰** BC31163  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   加入右單引號字元 (') 來分隔屬性值。
