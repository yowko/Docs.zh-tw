# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>命名空間中類型的完整限定的名稱不能超過&lt;數目&gt;字元
類型的完整限定名稱不能超過指定的字元數。  
  
 **錯誤 ID︰** BC30031  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請縮短類型的名稱。
