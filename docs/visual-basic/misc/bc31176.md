# XML 常值中不允許 &#39;character&#39; 字元 (0x&lt;十六進位&gt;)
已在 XML 常值中使用不支援的字元。  
  
 **錯誤 ID︰**BC31176  
  
### 更正這個錯誤  
  
-   請移除不支援的字元。
