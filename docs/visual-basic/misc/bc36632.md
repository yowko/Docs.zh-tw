# 可為 Null 的參數必須指定類型
參數包含可為 Null 的類型修飾詞 \(?\)，但未指定類型。  
  
 **錯誤 ID︰**BC36632  
  
### 更正這個錯誤  
  
-   宣告可為 Null 的參數時，請指定類型。
