# 無法將 &#39;&lt;類型名稱&gt;&#39; 當做屬性使用，因為它沒有 &#39;System.AttributeUsageAttribute&#39; 屬性
已嘗試使用的屬性，在宣告時未使用 `System.AttributeUsageAttribute` 來定義其使用方式。  
  
 **錯誤 ID︰**BC31505  
  
### 更正這個錯誤  
  
1.  自訂屬性必須是衍生自`System.Attribute` 的類別，並且已套用 `AttributeUsageAttribute` 屬性。
