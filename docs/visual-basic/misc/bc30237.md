# 參數已使用名稱 &#39;&lt;引數名稱&gt;&#39; 進行宣告
程序宣告以相同名稱定義兩個引數。 在程序宣告中的所有引數名稱必須是唯一的。  
  
 **錯誤 ID：**BC30237  
  
### 更正這個錯誤  
  
-   變更其中一個引數的名稱。
