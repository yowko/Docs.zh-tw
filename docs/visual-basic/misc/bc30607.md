# &#39;NotInheritable&#39; 類別不可以有宣告為 &#39;&lt;規範名稱&gt;&#39; 的成員
覆寫修飾詞不能與 `NotInheritable` 類別搭配使用，因為無法覆寫它們的成員。  
  
 **錯誤 ID：**BC30607  
  
### 更正這個錯誤  
  
-   請從類別定義中移除覆寫修飾詞 \(例如 `Overridable`、`NotOverridable` 或 `MustOverride`\)。
