# 無法讀取傳遞給 TextFieldParser 的資料流
[TextFieldParser Object](../../visual-basic/language-reference/objects/textfieldparser-object.md) 無法讀取已傳遞給它的資料流。 這可能是因為嘗試讀取不是文字檔的檔案。  
  
### 更正這個錯誤  
  
-   確定檔案是文字檔。
