# 即時運算視窗中的 &#39;GoTo&#39; 陳述式無效
`GoTo` 陳述式會執行分支，但在偵錯內容中則不允許執行。  
  
 **錯誤 ID︰**BC30133  
  
### 更正這個錯誤  
  
-   請不要在 \[即時運算\] 視窗中發出 `GoTo` 陳述式。
