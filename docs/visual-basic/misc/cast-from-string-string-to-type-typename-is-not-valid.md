# 從字串 &quot;&lt;字串&gt;&quot; 到類型 &#39;&lt;類型名稱&gt;&#39; 的轉換無效
嘗試將字串變數轉換成另一種類型不正確。  
  
### 更正這個錯誤  
  
1.  請檢查轉換中的類型，並確定您嘗試轉換成有意義的類型。
