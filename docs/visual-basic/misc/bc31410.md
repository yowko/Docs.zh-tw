# 多個基底介面中宣告的多載化方法無效
多個繼承介面隱含地多載相同方法。  
  
 **錯誤 ID︰**BC31410  
  
### 更正這個錯誤  
  
-   請使用 `Shadows` 修飾詞，而不要使用 `Overloads` 修飾詞。
