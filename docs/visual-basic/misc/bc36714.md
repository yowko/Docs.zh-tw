# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="28f6b-102">無法初始化展開的屬性</span><span class="sxs-lookup"><span data-stu-id="28f6b-102">Expanded properties cannot be initialized</span></span>
<span data-ttu-id="28f6b-103">您可以將自動實作的屬性當做其宣告的一部分來初始化，但無法初始化展開的屬性。</span><span class="sxs-lookup"><span data-stu-id="28f6b-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="28f6b-104">**錯誤 ID:** BC36714</span><span class="sxs-lookup"><span data-stu-id="28f6b-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="28f6b-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="28f6b-105">To correct this error</span></span>  
  
- <span data-ttu-id="28f6b-106">使用自動實作的屬性，或從屬性宣告中移除初始化。</span><span class="sxs-lookup"><span data-stu-id="28f6b-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="28f6b-107">範例</span><span class="sxs-lookup"><span data-stu-id="28f6b-107">Example</span></span>  
 <span data-ttu-id="28f6b-108">下列範例會顯示自動實作的屬性和展開的屬性。</span><span class="sxs-lookup"><span data-stu-id="28f6b-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="28f6b-109">您可以使用指派或 `New` 子句來初始化自動實作的屬性，但無法初始化展開的屬性。</span><span class="sxs-lookup"><span data-stu-id="28f6b-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
