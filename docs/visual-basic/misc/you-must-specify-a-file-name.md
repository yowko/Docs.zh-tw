# 您必須指定檔名。
您已嘗試執行需要提供檔名的作業 \(例如移動或複製檔案\)。  
  
### 更正這個錯誤  
  
-   提供所需的檔名。  
  
-   確定路徑的結尾不是目錄分隔符號字元 \(\\\)。
