# 沒有縮小轉換這些引數呼叫: &lt;清單&gt;，就沒有可存取的多載 &#39;&lt;方法名稱&gt;&#39;
已呼叫多載方法，但沒有方法可在未縮小轉換的情況下，符合所提供的引數清單。  
  
### 更正這個錯誤  
  
1.  指定 `Option``Strict` `Off`。  
  
2.  變更引數，以符合多載方法的簽章。
