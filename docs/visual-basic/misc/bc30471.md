# 運算式不是陣列或方法，所以不可以有引數清單
您已為不是陣列或方法的運算式提供引數清單。 只有陣列或方法才能具有引數清單。  
  
 **錯誤 ID︰**BC30471  
  
### 更正這個錯誤  
  
-   請移除引數清單。
