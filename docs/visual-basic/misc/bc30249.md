# 必須是 &#39;=&#39; (宣告)
`Mid` 陳述式或 `#Const` 編譯指示詞不包含值的指派。  
  
 **錯誤 ID︰**BC30249  
  
### 更正這個錯誤  
  
-   請加入等號 \(`=`\) 以及要指派給字串或編譯常數的值。
