# 沒有 &#39;As&#39; 子句的變數宣告; 假設是 Object 的類型
變數宣告未指定 `As` 子句。  
  
 `As` 子句會識別要與程式設計項目相關聯的資料類型。 在 [Dim Statement](../../visual-basic/language-reference/statements/dim-statement.md) 中，它會指定一個或多個變數的資料類型。 如果您未在 `Dim` 陳述式中包含 `As` 子句，變數的資料類型會預設為 `Object`。  
  
 根據預設，這個訊息是一個警告。 如需隱藏警告或將警告視為錯誤的相關資訊，請參閱[在 Visual Basic 中設定警告](/visual-studio/ide/configuring-warnings-in-visual-basic)。  
  
 **錯誤 ID︰**BC42020  
  
### 更正這個錯誤  
  
-   請在 `Dim` 陳述式中包含 `As` 子句，以指定變數的資料類型。
