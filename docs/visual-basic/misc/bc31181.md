# 無效的屬性值; 必須是 &#39;value&#39;
XML 文件常值中指定了無效的屬性值。 XML 文件常值的有效屬性值是以 XML 1.0 規格所指定。  
  
 **錯誤 ID︰**BC31181  
  
### 更正這個錯誤  
  
-   根據 XML 1.0 規格所指定，將屬性值更新為有效的值。
