# &#39;SyncLock&#39; 運算元不能屬於類型 &#39;&lt;類型名稱&gt;&#39;，因為 &#39;&lt;類型名稱&gt;&#39; 不是參考類型
`SyncLock` 陳述式允許在單一運算式上同步處理陳述式，方法是確定多個執行緒不會同時執行相同的陳述式。`SyncLock` 陳述式中的運算式類型必須是參考類型 \(例如類別、模組、介面、陣列或委派\)。  
  
 **錯誤 ID︰**BC30582  
  
### 更正這個錯誤  
  
-   請將類型變更為適當的參考類型。
