# 引數 &#39;&lt;引數名稱&gt;&#39; 不是一個有效的值
引數的值無效。  
  
### 更正這個錯誤  
  
1.  請檢查引數的值。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
