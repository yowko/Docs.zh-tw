# &#39;System.Runtime.InteropServices.DllImportAttribute&#39; 無法套用至非空白主體的 Sub、Function 或 Operator
`DllImportAttribute` 屬性已套用至不是空的 `Sub`、`Function` 或 `Operator`。  
  
 **錯誤 ID︰**BC31522  
  
### 更正這個錯誤  
  
-   請移除 `Sub`、`Function` 或 `Operator` 中的所有程式碼，以使用這個屬性。
