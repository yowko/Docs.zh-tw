# 在此內容無法存取方法 &#39;&lt;方法&gt;&#39; 的定義
不存在的彙總函式已隨附於 `Into` 運算子。  
  
 **錯誤 ID︰**BC36594  
  
### 更正這個錯誤  
  
-   請檢查彙總函式名稱的拼字，並進行任何必要的更正。  
  
-   請將方法以自訂彙總函式 \(擴充方法\) 加入。 如需建立自訂彙總函式的詳細資訊，請參閱[Aggregate Clause](../../visual-basic/language-reference/queries/aggregate-clause.md)。
