# 屬性不可以宣告為 &#39;&lt;modifiername&gt;&#39;
`Property` 宣告中所使用的語法發生錯誤。 這個錯誤通常表示指定的關鍵字 \(例如 `WithEvents`\) 不能用在 `Property` 宣告中。  
  
 **錯誤 ID︰**BC30639  
  
### 更正這個錯誤  
  
-   請檢查 `Property` 宣告的語法。
