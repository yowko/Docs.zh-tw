# 無法以 &#39;MyClass&#39; 呼叫 &#39;MustOverride&#39; 方法 &#39;&lt;方法名稱&gt;&#39;
`MyClass` 相當於 `Me`，但其上的所有方法引動過程都會當做所要叫用的方法為 `NotOverridable`。  
  
 **錯誤 ID：**BC30614  
  
### 更正這個錯誤  
  
-   移除 `MustOverride` 修飾詞，或是在基底類別中宣告此方法，然後繼承並覆寫該類別。
