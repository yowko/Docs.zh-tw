# <a name="modifier-is-not-valid-on-a-method-declaration"></a>'\<修飾詞 >' 在方法宣告中無效
`Function` 或 `Sub` 陳述式包含無效的關鍵字。  
  
 **錯誤 ID:** BC30242  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從 `Function` 或 `Sub` 陳述式中移除無效的關鍵字。
