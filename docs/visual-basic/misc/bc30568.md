# 陣列初始設定式擁有 &lt;數字&gt; 個項目，項目太多
您的陣列初始設定式包含太多項目。  
  
 **錯誤 ID︰**BC30568  
  
### 更正這個錯誤  
  
-   使用 `ReDim` 變更陣列的大小。
