# 多載解析失敗，因為沒有 &#39;&lt;方法&gt;&#39; 是可存取的
您已呼叫多載的方法，但編譯器無法使用相符參數數目的參數清單來找到可存取的方法。  
  
 **錯誤 ID︰**BC30517  
  
### 更正這個錯誤  
  
1.  請檢查您提供的引數清單。  
  
2.  請檢查您嘗試呼叫的方法來判斷是否可供存取。
