# 結構 &#39;&lt;結構名稱&gt;&#39; 至少必須包含一個執行個體成員變數或 Event 宣告
會出現功能性空白結構。 在 `Structure` 陳述式與 `End Structure` 陳述式之間至少必須宣告一個變數成員。  
  
 **錯誤 ID︰**BC30281  
  
### 更正這個錯誤  
  
-   將 `Dim` 陳述式或 `Event` 陳述式加入結構中。
