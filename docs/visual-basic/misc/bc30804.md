# 不再支援 &#39;Variant&#39; 類型。請使用 &#39;Object&#39; 類型代替
`Object` 資料類型取代 `Variant`。  
  
 **錯誤 ID︰**BC30804  
  
### 更正這個錯誤  
  
-   請將項目宣告為 `Object`。
