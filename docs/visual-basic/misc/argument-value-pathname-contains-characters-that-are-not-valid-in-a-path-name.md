# 引數值 &#39;&lt;路徑名稱&gt;&#39; 包含不得出現於路徑名稱的字元
路徑名稱包含一個或多個無效字元。  
  
### 更正這個錯誤  
  
-   請從路徑名稱中移除該字元。
