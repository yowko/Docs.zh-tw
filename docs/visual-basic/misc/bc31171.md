# XML 註解中不可有字元順序 &#39;--&#39;
XML 註解常值包含字元順序 '\-'，不支援此情形。  
  
 **錯誤 ID︰**BC31171  
  
### 更正這個錯誤  
  
-   請從 XML 註解常值移除字元順序 '\-'。
