# <a name="protected-types-can-only-be-declared-inside-of-a-class"></a>只可以在類別內宣告 Protected 類型
某個類型已在模組內宣告為 `Protected`。  
  
 **錯誤 ID︰** BC31047  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  在類別內宣告此類型。  
  
2.  請移除 `Protected` 修飾詞。
