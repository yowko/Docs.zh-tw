# 泛型方法不可以公開至 COM
包含一或多個泛型程序的 [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] 元件匯出至 COM 元件。  
  
 元件物件模型 \(COM\) 不支援泛型類型，而且無法與它們交互操作。  
  
 **錯誤 ID︰**BC30943  
  
### 更正這個錯誤  
  
-   請從 [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] 元件中移除泛型程序，或不要將它用於 COM Interop。
