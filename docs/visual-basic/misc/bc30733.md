# &#39;&lt;程序名稱&gt;&#39; 已宣告 &#39;&lt;成員名稱&gt;&#39;，前者是針對這個 &lt;容器名稱&gt; 所產生
已宣告成員的名稱與 Comom Language Runtime 所宣告的成員相同。  
  
 **錯誤 ID︰**BC30733  
  
### 更正這個錯誤  
  
-   請變更要移除衝突的成員名稱。
