# 建構函式必須宣告為 Sub，而不是 Function
您已嘗試宣告 `Function New`。 建構函式必須宣告為 `Sub New`。  
  
 **錯誤 ID︰**BC30493  
  
### 更正這個錯誤  
  
-   使用 `Sub` 取代 `Function`。
