# <a name="line-number-cannot-be-read-because-it-exceeds-the-maximum-line-size"></a>行\<數字 > 無法讀取，因為它超過最大線條大小。
無法讀取這行，因為超過行數上限。 這通常表示檔案中有左引號，但沒有右引號。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請在檔案中插入右引號。
