# 不支援 Get (唯寫屬性)
由於這是唯寫屬性，因此無法加以讀取。  
  
### 更正這個錯誤  
  
1.  請檢查屬性，並判斷在哪些情況下可以設定屬性。  
  
2.  移除屬性的參考。
