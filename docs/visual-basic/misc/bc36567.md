# 參數 &#39;&lt;參數名稱&gt;&#39; 已經有符合的引數
在相同的程序呼叫中，依名稱多次傳遞引數。  
  
 **錯誤 ID︰**BC36567  
  
### 更正這個錯誤  
  
-   請確認是否拼錯其中一個引數名稱。  
  
-   請從引數清單中移除重複引數。
