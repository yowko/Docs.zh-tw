# <a name="project-already-has-a-reference-to-assembly-assemblyidentity"></a>專案已經有組件的參考\<組件識別 >
專案已經有組件的參考\<組件識別 >。 第二個參考 '\<檔案路徑 >' 無法加入。  
  
 專案多次參考相同的組件。  
  
 組件識別包含組件的名稱、版本、公開金鑰 (如果有的話) 和文化特性。  
  
 這個錯誤的其中一個可能原因是另一份組件的參考，而且是透過與原始參考之檔案路徑不同的檔案路徑。  
  
 **錯誤 ID:** BC32208  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除第二個參考。 它是不需要的，因為它參考相同的組件。
