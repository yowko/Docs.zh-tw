# &#39;&lt;引數名稱&gt;&#39; 引數必須是在 -32768 到 65535 的範圍中
引數的值超出允許的範圍 \-32,768 到 65,535 之間。  
  
### 更正這個錯誤  
  
1.  請確定引數的值落在有效範圍內。  
  
2.  如果透過計算來提供引數，請檢查計算所處理的值。
