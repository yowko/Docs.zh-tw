## <a name="an-anonymous-type-definition"></a><span data-ttu-id="03da5-180">匿名型別定義</span><span class="sxs-lookup"><span data-stu-id="03da5-180">An Anonymous Type Definition</span></span>  
 <span data-ttu-id="03da5-181">為了回應匿名型別的實例宣告, 編譯器會建立新的類別定義, 其中包含指定的屬性。</span><span class="sxs-lookup"><span data-stu-id="03da5-181">In response to the declaration of an instance of an anonymous type, the compiler creates a new class definition that contains the specified properties.</span></span>
