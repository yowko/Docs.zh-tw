# My.Computer.FileSystem Object
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

提供用於處理磁碟機、檔案和目錄的屬性和方法。  
  
## 備註  
 如需有關的方法和屬性的資訊`My.Computer.FileSystem`物件，請參閱<xref:Microsoft.VisualBasic.FileIO.FileSystem>。  
  
 如需詳細資訊，請參閱 [File Access with Visual Basic](../../../visual-basic/developing-apps/programming/drives-directories-files/file-access.md)。  
  
> [!NOTE]
>  您也可以使用中的類別<xref:System.IO?displayProperty=fullName>若要使用的磁碟機、 檔案和目錄的命名空間。  
  
## 需求  
 **命名空間︰** <xref:Microsoft.VisualBasic.MyServices>  
  
 **類別:** <xref:Microsoft.VisualBasic.MyServices.FileSystemProxy> \(提供存取<xref:Microsoft.VisualBasic.FileIO.FileSystem>\)  
  
 **組件**：Visual Basic 執行階段程式庫 \(在 Microsoft.VisualBasic.dll 中\)
