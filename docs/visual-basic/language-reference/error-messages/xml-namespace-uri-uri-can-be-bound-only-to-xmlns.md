# <a name="xml-namespace-uri-39httpwwww3orgxml1998namespace39-can-be-bound-only-to-39xmlns39"></a>XML 命名空間 URI &#39; http://www.w3.org/XML/1998/namespace &#39;可以為繫結 &#39; xmlns &#39;
XML 命名空間宣告中使用 URI http://www.w3.org/XML/1998/namespace。 這個 URI 是保留的命名空間，不能包含在 XML 命名空間宣告。  
  
 **錯誤 ID:** BC31183  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   移除 XML 命名空間宣告或取代為有效的命名空間 URI 的 URI http://www.w3.org/XML/1998/namespace。
