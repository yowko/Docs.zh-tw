# TextFieldParser is unable to complete the read operation because maximum buffer size has been exceeded
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

無法完成作業，因為已經超過緩衝區大小的上限 \(10,000,000 個位元組\)。  
  
### 若要更正這個錯誤  
  
-   確定檔案中沒有錯誤的欄位。
