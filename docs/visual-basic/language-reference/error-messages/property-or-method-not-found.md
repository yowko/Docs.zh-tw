# Property or method not found
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

未定義參考的物件方法或物件屬性。  
  
### 若要更正這個錯誤  
  
-   您可能拼錯物件的名稱。  若要查看已為物件定義的屬性和方法，請顯示物件瀏覽器。  選取適當的物件程式庫，以檢視可用之屬性 \(Property\) 和方法的清單。
