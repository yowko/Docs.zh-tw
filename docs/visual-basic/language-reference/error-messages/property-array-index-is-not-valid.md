# Property array index is not valid
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

提供的值對於屬性 \(Property\) 陣列索引而言無效。  
  
### 若要更正這個錯誤  
  
-   請查閱元件的參考文件，以確認您的索引是否在指定屬性 \(Property\) 的有效範圍之內。
