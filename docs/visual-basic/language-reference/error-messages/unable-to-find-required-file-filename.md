# Unable to find required file &#39;&lt;filename&gt;&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Visual Studio 所需要的某個檔案已經遺失或損毀。  
  
 **錯誤 ID：**BC30655  
  
### 若要修正這個錯誤  
  
-   請重新安裝 Visual Studio。
