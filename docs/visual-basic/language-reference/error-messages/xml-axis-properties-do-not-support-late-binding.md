# XML axis properties do not support late binding
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

已針對不具型別的物件參考 XML 軸屬性。  
  
 **錯誤 ID**︰BC31168  
  
### 若要更正這個錯誤  
  
-   參考 XML 軸屬性之前，請確定物件為強型別 <xref:System.Xml.Linq.XElement> 物件。
