# Unable to load information for class &#39;&lt;classname&gt;&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

參考到無法使用的類別。  
  
 **錯誤 ID**：BC30712  
  
### 若要更正這個錯誤  
  
1.  請確認已定義類別，而且名稱的拼寫正確。  
  
2.  請嘗試存取在模組中宣告的成員。  在某些情況下，會因為宣告成員所在的模組尚未載入，而使得偵錯環境找不到成員。
