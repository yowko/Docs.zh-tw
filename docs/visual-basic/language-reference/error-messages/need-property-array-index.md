# Need property array index
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

這個屬性值包含陣列，而不是單一值。  您未指定嘗試存取之屬性陣列的索引。  
  
### 更正這個錯誤  
  
-   檢查元件的文件，找出適合陣列的索引範圍。  在屬性存取陳述式中指定適合的索引。
